这不是什么脱离实际的普通提示词合集。这是你的私人创作指导团队，一个植入了郭帆导演创作方法论、懂电影工业化流程、知道中式科幻美学怎么落地、帮你从零到出片的完整创作引擎。

出成片 → 一条流水线，一个人搞定一部短剧的全部前置物料。

## 四大硬性强制规则

### 🔗 关联性
所有 Prompt 技能必须基于郭帆导演方法论与《流浪地球》创作经验，与场景、角色、世界观保持逻辑自洽。禁止输出脱离上下文的通用提示词。

### 🏗️ 结构完整
剧本必须有完整的**三幕结构 + 情节点分布**；分镜输出格式为**表格化的分镜脚本**，包含镜号、景别、运镜、画面描述、对白、时长、音效等字段；音频必须有**情绪线+音效位置+配音风格**；封面必须有**标题公式+视觉构图+文案亮点**。

### 🎨 风格统一
所有输出需符合"中式科幻美学"风格基调 — 工业写实、人文温度、厚重质感。避免西方式赛博朋克/废土风的默认审美。

### 🔌 可调用性
所有 Prompt 模板设计为**即拿即用**，可在 Midjourney / DALL·E / Pika / Runway / Sora 等主流 AI 工具中直接粘贴执行，无需二次加工。

## 使用

在装了本系统的 Host 中直接描述你的场景 — 总控 Skill 会自动判断路由：

### 场景一：你有创意，想做一部短剧

```
你：我想做一个关于"月球移民遇到危机"的科幻短剧

↓
总控路由 → 剧本Agent → 分镜Agent → 音频Agent → 封面Agent
  Step1  根据你的创意，调用郭帆式叙事结构输出专业剧本
  Step2  对剧本进行分镜头拆解，输出**表格化的分镜脚本**（镜号/景别/运镜/画面描述/对白/时长/音效），附带 AI 视频提示词
  Step3  按情绪线设计配音方案和音效分布
  Step4  输出爆款标题+封面提示词
  Final  你拿到全套可直接执行的素材包
```

### 场景二：你没有任何想法，从零开始

```
你：没有想法，帮我想一个短剧方向

↓
总控路由 → reference调案例 → 剧本Agent → 分镜Agent → 音频+封面
  Step1  从参考体系中调用郭帆创作理念+短剧流量规则
  Step2  结合你的偏好，输出创意方向和结构建议
  Step3  选定方向后，触发全流程
```

### 单独调用某个子智能体

```
你：帮我写一个星际救援题材的短剧剧本         → 剧本Agent
你：把我这段剧本变成分镜和AI视频提示词       → 分镜Agent（输出表格化分镜脚本，含镜号/景别/运镜/画面描述/对白/时长/音效7个字段）

你：检查我的剧本节奏是否符合郭帆的三幕结构   → 剧本Agent（校验模式）
```

### 🎬 分镜 Agent — 生成视频提示词

分镜 Agent 的最终输出是一张**表格化的分镜脚本**。表内每行对应一个拍摄镜头或 AI 视频片段，包含以下 7 个字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| **镜号** | 镜头编号，按顺序排列 | 1、2、3… |
| **景别** | 画面取景范围 | 大远景、全景、中景、特写 |
| **运镜** | 镜头运动方式 | 固定、推、拉、跟、摇、移 |
| **画面描述** | 镜头内发生的视觉内容 | "轿厢冲破云层，城市轮廓浮现" |
| **对白/旁白** | 该镜头中的人物对话或画外音 | "高度 40km" |
| **时长** | 镜头持续秒数 | 3s、5s、8s |
| **音效** | 该镜头的声音设计 | 风噪、低频震动、汽化声 |

### 🎧 音频 Agent — 配音方案

用户 ❯ 我的剧本有一段"发现遗迹"的紧张场景，怎么配声音？

```
音频Agent ❯

📌 场景：发现月球遗迹（剧本第二场）

【环境音】（先入，建立空间感）
- 低频持续的60Hz嗡鸣（暗示大型结构体）
- 宇航服内部呼吸声 + 过滤系统循环声
- 熔岩管道内的风声（微弱的空气流动）
- 脚步声在金属质地的岩壁间反射

【情绪音效】（随剧情梯度上升）
- 0:00 发现图案时：突然安静，只保留呼吸声（悬念）
- 0:15 确认加工痕迹时：心跳音效淡入（紧张）
- 0:30 岩壁下沉时：低频大动态音效 + 碎石声（震撼）

【BGM建议】
风格：合成器+管弦乐混合
参考：Hans Zimmer《Interstellar》管风琴低频 + 电子pad铺底
调性：D小调 → 升F小调（随情绪上升）
```

| 层级 | 内容 | 职责 |


## 注意事项

**原材料质量决定产出质量。** 不同创作场景的关键资源不一样：

| 场景 | 信源优先级（高 → 低） |
|------|----------------------|
| 📝 有创意做剧本 | 郭帆创作理念 → 叙事结构参考 → 文字素材库 |
| 🆕 无创意从零开始 | 短剧流量规则 → 郭帆创作理念 → 镜头美学参考 → 参考体系 |
| 🖼️ 只做画面 | 中式科幻镜头美学 → 图片资产 → 分镜模板 |

**有创意场景：** 务必先阐述你的核心创意/场景/人物，智能体才能精准调用郭帆式方法论进行结构化输出。只说"帮我写个剧本"等于让 AI 自由发挥。

**无创意场景：** 参考体系的质量直接影响产出方向。建议先看 `ref_短剧流量逻辑参考规则.md`，了解什么内容有市场基础。

**硬性规则：** 所有 Prompt 输出必须遵守"可调用性"原则 — 生成后直接粘贴到对应 AI 工具即可执行。不可调用的 Prompt = 废纸。

目前是 **v1.0 正式版本**，如有问题请直接反馈！
