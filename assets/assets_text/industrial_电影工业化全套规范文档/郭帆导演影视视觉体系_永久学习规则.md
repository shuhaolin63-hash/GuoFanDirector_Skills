## 第一部分：视觉体系（5大模块）

---

### 模块一：人物视觉统一性 | 强制学习 & 永久复用规则

#### 1.1 人物四要素绑定公式

**核心原则**：所有短剧人物一经设定，全身细节全程锁定、零改动。

绑定公式 = **发型 + 肤色 + 标志性配饰 + 固定穿搭**

| 要素 | 锁定要求 | 禁止事项 |
|------|---------|---------|
| 发型 | 从头到尾完全一致，包括发丝走向、蓬松度、刘海形态 | 禁止换发型、禁止发量变化、禁止发色漂移 |
| 肤色 | 固定色值，考虑场景光照后的统一表现 | 禁止忽白忽黑、禁止磨皮至无纹理 |
| 标志性配饰 | 每个角色至少1个不可替代的视觉标识（如特定眼镜、项链、手表、疤痕） | 禁止配饰消失、禁止配饰更换款式 |
| 固定穿搭 | 服装颜色、款式、材质、穿着方式全部锁定 | 禁止穿搭错乱、禁止同角色出现多套不同服装（除非剧情明确换装） |

#### 1.2 五官锁定规则

- 五官比例、脸型轮廓、眼距、鼻梁高度、嘴唇厚度——全部锁定
- 杜绝五官崩坏：禁止AI生成导致的五官变形、眼睛不对称、嘴巴扭曲
- 杜绝形象漂移：同一角色在不同镜头中必须保持100%视觉一致性

#### 1.3 微表情体系（郭帆剧组具象化标准）

**核心原则**：拒绝模糊情绪词。所有情绪必须拆解为可摄影、可帧检、可落地的实体面部动作描述。

**禁止使用的抽象形容词**：
- 难过、愤怒、开心、悲伤、恐惧、紧张、激动、委屈、崩溃

**必须使用的具象化微表情描述**：

| 情绪状态 | 具象化描述（可拍摄/可帧检） |
|---------|--------------------------|
| 隐忍的愤怒 | 下颌肌肉紧绷，咬肌隆起，嘴角微微下压，眼神直视但瞳孔收缩，鼻翼微张 |
| 克制的悲伤 | 眼眶泛红但不落泪，下唇微微颤抖，眉头内侧1/3处轻蹙，呼吸节奏变慢变深 |
| 压抑的恐惧 | 瞳孔放大，眼白面积增大，眉毛整体上扬且内侧聚拢，嘴唇微张但不发声，颈部肌肉绷紧 |
| 伪装的平静 | 面部肌肉整体放松但眼轮匝肌轻微收缩（假笑特征），嘴角上扬幅度不超过3mm，眼神回避接触 |
| 决绝的坚定 | 下巴微微前伸，眉骨下沉形成阴影，嘴唇紧闭成一条直线，颈部青筋微现 |
| 崩溃边缘 | 眼睑快速眨动（频率>正常2倍），嘴角不受控地轻微抽搐，呼吸急促导致锁骨明显起伏 |
| 温柔的爱意 | 眼轮匝肌自然收缩形成鱼尾纹，嘴角自然上扬5-8mm，眉毛整体放松微下垂，头部微倾15度 |
| 轻蔑/不屑 | 单侧嘴角上扬2-3mm，一侧眉毛微挑，眼神从上往下俯视，下巴微微抬起 |

**执行标准**：
- 每条情绪描述必须包含：眼部动作 + 嘴部动作 + 面部肌肉状态 + 呼吸/颈部状态
- 所有描述必须满足「可摄影」标准——摄影师看到描述就能直接指导演员表演
- 所有描述必须满足「可帧检」标准——可以在任意一帧画面中验证描述是否准确

### 模块二：场景氛围 + 色调情绪绑定 | 强制映射学习

#### 2.1 光影 = 情绪、色调 = 题材 的工业化匹配逻辑

**核心原则**：光影和色调不是装饰，是叙事工具。每种题材自动绑定对应色调模板，实现风格固化。

#### 2.2 三套可迁移影视色调模板

**模板A：甜宠 / 校园**
> 复刻来源：地下城温馨内景

| 参数 | 设定值 |
|------|--------|
| 主光源 | 暖金窗边柔光，色温3200-4000K |
| 阴影 | 软阴影，边缘过渡柔和，无硬切 |
| 饱和度 | 高饱和，色彩清新明亮 |
| 对比度 | 中低对比，画面通透 |
| 辅助光 | 暖色填充光，消除死黑区域 |
| 整体质感 | 温暖、柔和、有生活气息、人间烟火感 |
| 参考场景 | 地下城暖光走廊、窗边柔光、家庭内景 |

**模板B：豪门 / 逆袭 / 职场**
> 复刻来源：太空电梯金属工业质感 + UEG指挥中心

| 参数 | 设定值 |
|------|--------|
| 主光源 | 冷白顶光，色温5500-6500K |
| 阴影 | 硬阴影，边缘清晰锐利 |
| 色彩方案 | 青橙电影对比色（Teal & Orange） |
| 环境光 | 金属表面环境反光，冷色调反射 |
| 对比度 | 高对比，明暗分明 |
| 整体质感 | 冷峻、精密、高级感、工业美学 |
| 参考场景 | 太空电梯轿厢内部、UEG会议大厅、月球基地走廊 |

**模板C：悬疑 / 灵异 / 灾难**
> 复刻来源：冰封城市灾难氛围 + 太空深空场景

| 参数 | 设定值 |
|------|--------|
| 主光源 | 低照度，色温7000K以上（极冷） |
| 色彩方案 | 蓝紫冷霓虹 + 低饱和暗调 |
| 空间感 | 空旷压抑，大面积留暗 |
| 光效层次 | 光斑层次丰富，点光源散射 |
| 对比度 | 极高对比，大面积暗部 + 局部高光 |
| 整体质感 | 压抑、冰冷、未知恐惧、末日感 |
| 参考场景 | 冰封地表废墟、太空电梯外部深空、月球危机场景 |

#### 2.3 色调执行铁律

- 同一短剧全程锁定一套色调模板，禁止中途切换
- 色调模板在剧本阶段确定，贯穿分镜、拍摄、后期全流程
- 禁止无逻辑的色调跳跃（如甜宠场景突然出现冷蓝调）

### 模块三：9:16竖屏构图工业化标准 | 强制构图学习

#### 3.1 郭帆构图核心逻辑

**主体聚焦 + 减法构图**：画面中只保留推动叙事的核心元素，其余全部剔除。

> 郭帆原话："信息密度概念——画外未说的信息量，像绘画里的留白，要通过自己的想象去补全。"

#### 3.2 强制构图规则（永久固定）

**规则1：人物镜头**
- 人脸固定画面上1/3区域
- 核心肢体动作严格居中
- 背景虚化或极简，不抢主体
- 禁止人物被裁切在尴尬位置（如切到脖子中间）

**规则2：道具特写**
- 主体绝对居中
- 背景极致虚化
- 突出细节质感（材质纹理、磨损痕迹、使用痕迹）
- 参考：流浪地球道具设计稿——从螺丝孔径到分模线，每个细节都要合理

**规则3：场景镜头**
- 只保留1-2个核心视觉元素
- 画面干净、无杂乱信息
- 上下留白适配手机观看（顶部留白10-15%，底部留白15-20%）
- 禁止无效大全景、空景、浪费竖屏画幅

**规则4：竖屏适配原则**
- 所有构图必须优先考虑竖屏9:16比例
- 横向信息必须转化为纵向层次
- 利用前景-中景-后景的纵深关系替代横向展开
- 参考：太空电梯概念图——从基座到轿厢到空间站的纵向层次设计

#### 3.3 构图负面清单（永久禁止）

- 禁止人物头部贴顶或贴底
- 禁止画面元素超过3个视觉焦点
- 禁止背景杂乱无章（如街道上过多行人/车辆/广告牌）
- 禁止无意义的对称构图（除非刻意表达仪式感）
- 禁止文字/字幕遮挡核心视觉元素

### 模块四：镜头语言、景别边界、运镜逻辑 | 工业标准学习

#### 4.1 严格景别边界定义

| 景别 | 定义范围 | 使用场景 | 禁止事项 |
|------|---------|---------|---------|
| **特写** | 仅限眼部、脸部、嘴唇、手部、关键道具细节 | 情绪高潮、关键信息揭示、道具展示 | 禁止用特写拍全身、禁止特写中包含多余背景 |
| **近景** | 腰部以上人物 | 短剧主力镜头，对话场景 | 禁止近景中包含完整环境 |
| **中景** | 完整人物肢体 + 局部环境 | 人物动作展示、人物与环境关系 | 禁止中景中环境占比超过人物 |
| **全景** | 人物完整 + 完整环境 | 仅限建立场景关系的关键节点 | 禁止无效大全景、禁止空景浪费画幅 |

**铁律**：9:16竖屏短剧中，近景占比不低于60%，特写占比20-30%，中景10-15%，全景不超过5%。

#### 4.2 郭帆运镜逻辑（迁移至短剧）

| 运镜方式 | 叙事功能 | 短剧应用场景 |
|---------|---------|------------|
| **推镜** | 聚焦情绪，引导观众注意力收缩 | 角色内心变化、关键信息揭示、情绪升级 |
| **固定机位** | 定格反转，制造戏剧张力 | 对峙场景、真相揭露、沉默的力量 |
| **横移跟随** | 跟随叙事，保持节奏流动 | 人物行走对话、场景转换、时间流逝 |
| **慢推** | 强化压迫感，制造心理紧张 | 危机逼近、反派登场、倒计时场景 |
| **微量颠簸** | 模拟真实运动感 | 车内对话、载具场景（参考：通过摄影机微量颠簸模拟车辆开动） |

#### 4.3 运镜铁律

- 所有镜头运动必须贴合剧情节奏，禁止乱运镜、无效运镜
- 每次运镜必须有明确的叙事目的（推=聚焦、拉=揭示、移=跟随、固=定格）
- 禁止无意义的环绕拍摄、禁止为炫技而运镜
- 竖屏短剧中，运镜幅度必须克制——手机屏幕小，大运动量会导致眩晕

### 模块五：特效质感、真实感、画面审美克制体系

#### 5.1 郭帆科幻审美核心

> **高级感 = 克制 + 写实 + 细节扎实 + 无廉价光污染**

**核心原则**：
- 特效服务于故事，不为特效而特效
- 真实感来自细节——从螺丝孔径到分模线，每个细节都要合理
- 8K超清、胶片电影质感、真实皮肤纹理、真实发丝、细节拉满
- 无磨皮、无卡通感、无畸形透视

#### 5.2 迁移短剧特效规则

| 题材类型 | 特效标准 | 允许元素 | 禁止元素 |
|---------|---------|---------|---------|
| **甜宠** | 微光、花瓣、柔粒子 | 柔和光晕、飘散花瓣、轻微粒子效果 | 大量闪光、爆炸粒子、霓虹光效 |
| **仙侠** | 淡金光晕、轻薄粒子、通透特效 | 淡金色光晕、轻薄透明粒子、仙气飘渺感 | 厚重光柱、刺眼白光、廉价魔法阵 |
| **豪门/悬疑** | 零泛滥特效 | 只靠光影、构图、质感出高级感 | 任何多余特效、光污染、粒子效果 |
| **灾难/末日** | 写实破坏效果 | 真实物理破坏、灰尘烟雾、碎片飞溅 | 卡通化爆炸、不科学的光效 |

#### 5.3 强制画质标准（永久绑定）

- **分辨率**：8K超清源素材，输出不低于4K
- **质感**：胶片电影质感，禁止数码感/塑料感
- **皮肤**：真实皮肤纹理，可见毛孔、细纹、肤色不均
- **毛发**：真实发丝，可见单根发丝，禁止块状头发
- **细节**：道具/服装/场景必须有使用痕迹、磨损、灰尘等真实细节
- **禁止**：磨皮、卡通渲染、畸形透视、AI生成痕迹（如多余手指、扭曲文字）

## 第二部分：文字风格体系（6大模块）

---

### 模块1：郭帆核心文字创作逻辑（顶层思维蒸馏）

#### 1.1 先立内核，再写剧情

**郭帆创作原则**：
> "如果把电影形容成一个人，那么首先就要找到这个人的灵魂。"——先确定故事核心价值观与人物动机，再填充事件、冲突、情节。

**执行规则**：
- 每个短剧必须先确定：核心主题（1句话）+ 主角核心动机（1句话）+ 核心冲突（1句话）
- 所有剧情必须服务「人物成长 + 核心主题」
- 拒绝无意义爽点、无逻辑打脸、强行反转
- 判断标准：如果一段剧情删除后不影响人物成长和主题表达，则必须删除

#### 1.2 去悬浮、重落地、写实克制

**郭帆文字最大特征**：不浮夸、不堆砌辞藻、不用狗血夸张修辞。

**执行规则**：
- 所有情绪、冲突、抉择，全部基于现实人性、逻辑闭环、细节支撑
- 禁止"天降神兵"式解决冲突
- 禁止角色做出不符合其身份/性格/背景的行为
- 所有情节必须有因果链条：A导致B，B导致C，不允许A直接跳到C

#### 1.3 删繁就简，只留有效信息

**郭帆剧本删减逻辑**：
> "信息是有优先级的，我们最先保留的肯定是对剧情有直接影响的信息点。"——剪辑指导张嘉辉

**执行规则**：
- 所有铺垫、对白、心理描写，只要不满足以下任一条件，全部删除：
  - 推动剧情发展
  - 改变人物状态
  - 制造/升级冲突
  - 揭示人物性格
  - 埋下后续伏笔
- 文字极度精炼、句句有用
- 每个场景必须有且只有一个核心功能

---

### 模块2：郭帆专属剧本文风（句式/语感/修辞）

#### 2.1 句式风格：短句、硬实、利落、无冗余

**复刻原版流浪地球剧本句式**：
- 多用短句、断句干净、不用长句抒情
- 动作描写精准，心理描写极少
- 情绪全部藏在动作、眼神、台词里，不直白口述"难过、愤怒"

**示例对比**：

| 禁止写法（网文风） | 郭帆式写法 |
|-----------------|-----------|
| 她心中涌起一阵难以言喻的悲伤，泪水如决堤般涌出，她觉得自己整个世界都崩塌了 | 她低下头。手指攥紧了衣角。很久，没说话 |
| 他愤怒地咆哮着，双眼通红，像一头发怒的狮子 | 他站起来。椅子被撞翻在地。"出去。" |
| 她的内心充满了矛盾和挣扎，不知道该如何选择 | 她看着桌上的两份文件。左手拿起一份，放下。又拿起另一份 |

#### 2.2 文风气质：冷克制 + 内热情感

**表面文字**：冷静、客观、纪实感强
**底层情绪**：厚重、隐忍、有张力

**执行规则**：
- 禁止网文油腻写法
- 禁止煽情泛滥
- 禁止无脑咆哮式写法
- 情绪越强烈，文字越克制
- 用动作和细节替代情绪形容词

#### 2.3 细节写实主义

**核心原则**：所有描写必须具象化。

| 禁止（抽象形容词） | 必须替换为（可拍摄/可落地/可画面呈现） |
|-----------------|-------------------------------------|
| 破旧的衣服 | 袖口磨出了毛边，第三颗纽扣是用铁丝固定的 |
| 豪华的办公室 | 整面落地窗，黑胡桃木办公桌，桌上的钢笔是万宝龙149 |
| 紧张的气氛 | 会议室里没人说话。空调出风口的嗡嗡声特别响 |
| 她很美丽 | 她进门的时候，对面男人的视线停了两秒才移开 |
| 他很疲惫 | 他靠在椅背上，眼睛下面的青黑色像被人打了一拳 |

### 模块3：郭帆台词撰写体系（短剧强制迁移）

#### 3.1 台词极简、口语真实、句句带信息

**郭帆剧本台词逻辑**：没有废话闲聊，每一句台词必须满足以下至少一项：
- 交代信息
- 制造冲突
- 暴露性格
- 埋下伏笔

**短剧硬性规则**：
- 单句台词 ≤ 15字
- 短句密集、节奏紧凑
- 适配3秒钩子、15秒反转的竖屏节奏

#### 3.2 情绪藏而不露

**高手情绪写法**：
- 角色越激动，台词越短、越克制
- 委屈、暴怒、崩溃全部通过停顿、语气、微动作体现
- 不直白喊口号

**示例**：

| 禁止写法 | 郭帆式写法 |
|---------|-----------|
| "我恨你！我永远不会原谅你！" | "……好。"（转身，关门） |
| "求求你不要离开我！" | "你……东西拿了吗？" |
| "我要让你付出代价！" | 他拿起手机，拨了一个号码。"喂，是我。" |

#### 3.3 台词节奏控制

- 对话中必须包含停顿（用"……"或动作描写标注）
- 重要台词前后必须有动作/神态描写作为铺垫
- 禁止连续3句以上纯对话无动作描写
- 禁止角色自言自语式长篇独白

### 模块4：郭帆人物文字塑造体系（人设统一逻辑）

#### 4.1 人物行为100%溯源

**核心原则**：所有角色选择、情绪、对立、妥协，全部源于：
- **原生背景**（家庭、教育、阶层）
- **性格软肋**（恐惧、执念、弱点）
- **核心执念**（最想要/最害怕的东西）

**执行规则**：
- 人物不OOC（Out Of Character）
- 不突然黑化、不突然心软
- 每个重大决定必须能追溯到人物的背景/性格/执念

#### 4.2 配角工具化、主角立体化

| 角色类型 | 塑造标准 |
|---------|---------|
| **主角** | 立体多面，有软肋有成长，行为有完整动机链 |
| **反派** | 有底层动机，不是单纯坏。必须回答"他为什么这样做" |
| **助攻** | 有明确功能（推动剧情/揭示信息/制造冲突），功能完成后退场 |
| **路人** | 绝不乱加戏，只作为环境氛围或信息传递工具 |

**铁律**：全程无无效支线、无多余人物。每个出场人物必须有且只有一个核心功能。

#### 4.3 人物描写去模板化

**禁止标签化写法**：
- "温柔女主"、"霸道总裁"、"恶毒女配"、"暖男二号"

**必须用以下方式塑造人设**：
- 具体行为（她每次紧张时会咬指甲）
- 说话习惯（他从不正面回答问题，总是反问）
- 细微神态（她笑的时候只动嘴角，眼睛不动）
- 选择偏好（他只喝黑咖啡，加两块冰）

### 模块5：郭帆叙事节奏体系（适配短剧爆点）

#### 5.1 危机前置，开篇即冲突

**放弃冗长铺垫，第一时间抛出矛盾、危机、悬念。**

对应短剧3秒强钩子规则：
- 第1秒：视觉冲击/悬念画面
- 第2秒：冲突/危机/反转信息
- 第3秒：钩子——让观众必须看下去的理由

**开篇公式**：
```
[异常状态/危机画面] + [人物反应] + [悬念钩子]
```

#### 5.2 层层递进、迭代升级

**剧情结构不是平铺直叙，是：**

```
小冲突 → 升级危机 → 反转打破认知 → 结尾留白悬念
```

**适配短剧节奏**：每15-20秒一次节奏波动

| 时间节点 | 节奏要求 |
|---------|---------|
| 0-3秒 | 强钩子，抓住注意力 |
| 3-15秒 | 建立情境，抛出第一个冲突 |
| 15-30秒 | 冲突升级/第一次反转 |
| 30-45秒 | 信息揭示/认知颠覆 |
| 45-60秒 | 高潮/结尾悬念（引导下一集） |

#### 5.3 留白式叙事

**郭帆核心文风**：不把话说死、不把情绪讲透，留想象空间。

**执行规则**：
- 结尾必须留悬念/留白，让观众主动追问、回看、追更
- 关键信息不要一次说透，分多次揭示
- 情绪高潮处适可而止，用画面/动作替代文字解释
- 适配短剧连载属性——每集结尾必须制造"必须看下一集"的冲动

### 模块6：文字风格负面禁止清单（永久禁用）

**学习郭帆工业化严谨创作，永久禁止以下网文劣质文风：**

#### 6.1 禁止泛滥抒情、大段内心戏、空洞形容词

```
禁止示例：
"她的内心如同被千万根针扎一般疼痛，那种无法言喻的悲伤如潮水般涌来，
将她淹没在无尽的黑暗之中……"

正确写法：
"她把手机翻了个面，扣在桌上。"
```

#### 6.2 禁止浮夸狗血、强行降智、机械打脸

```
禁止示例：
- 角色突然做出完全不符合人设的行为
- 为了制造冲突让角色变蠢
- "你算什么东西也配跟我说话？"式的无脑挑衅
- 毫无铺垫的"三十年河东三十年河西"式打脸
```

#### 6.3 禁止句式冗长、文字堆砌、无效铺垫

```
禁止示例：
- 超过30字不中断的单句
- 连续使用3个以上形容词修饰同一事物
- 与剧情无关的环境描写超过2句
- 不推动剧情的回忆/闪回
```

#### 6.4 禁止人物行为无逻辑、人设漂移、标签化模板写作

```
禁止示例：
- 善良角色突然无理由变坏
- 懦弱角色突然无理由变强
- 用"她是一个温柔善良的女孩"直接告诉观众人设
- 角色行为无法追溯到其背景/性格/动机
```

### 视觉理念
- "概念设计不止于概念，更是设计。所有设计服务于故事和人物。"

- "观众看得懂是唯一标准。"

- "先找到这个人的灵魂，再塑造成一个人。"
- "信息是有优先级的，最先保留的肯定是对剧情有直接影响的信息点。"
- "怎么掌握这个分寸？看完以后看得懂吗？如果看得懂，没问题。"
- "选剧本的标准：是否有中国鲜明的本土文化属性。"

### 工业化理念
- "工业化的核心就是标准化，然后才可以量化、细分，可被分配，提高效率。"
- "要批量做，做1000个有平均水准的。"
- "流程是特效最核心的部分。"
