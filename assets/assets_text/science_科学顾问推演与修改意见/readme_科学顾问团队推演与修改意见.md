## 二、各领域科学推演内容

### 2.1 天体物理组（负责人：苟利军、梁文杰）

#### 课题：如何炸毁月球？

**原始设定**：集合地球上所有核弹直接炸毁月球

**科学顾问推演结论**：**不可行。**

- 导致恐龙灭绝的小行星撞击（直径10公里）释放的能量远超人类所有核弹，但地球并未毁灭
- 能毁灭月球的能量又比小行星撞击高出**多个数量级**
- 人类核弹在月表爆炸，效果就像"人放了一个屁"

**最终解决方案：相控阵核爆**

经过与理论物理组的反复推敲，最终形成以下方案：
1. 在月球环形山**环状布置3000枚以上核弹**
2. 从环外开始**向内顺序引爆**
3. 通过相控阵原理控制能量传播方向
4. 能量聚焦于**300公里深的上月幔**，形成30米见方的高热区
5. 焦点区域温度达到**3亿度**
6. 引爆科幻设定中的**重核聚变**
7. 链式反应导致月球不断爆炸直至解体

> **梁文杰注**：这个方案在现实中不存在，但在《流浪地球2》的设定框架内是合理的。借此还向观众介绍了**相控阵**这一广泛应用的概念。

#### 关于核弹数量

- 科学顾问最初提出：**30000枚**
- 考虑到电影制作和视觉效果，调整为：**3000枚**
- 科学依据：美国曾做过核弹阵列计算，7颗核弹按阵列布置产生的定向毁伤力是单枚总当量的**5-8倍**

#### 关于月球碎片落到地球的视觉表现

- 科学顾问已做了详细的天体物理计算
- 碎片在大气层中燃烧形成大规模的"流星雨"效果
- 地球上的角色看到的是被大气层烧蚀后的碎片

### 2.2 人工智能组（负责人：王元卓）

#### 课题：如何在50年内实现MOSS级别的超级AI？

**科学顾问提出的路径：人在回路（Human-in-the-loop）**

- 当前AI在某些专业领域（如围棋）已超越人类
- 但在**泛化领域**，"最后一公里"仍无法超越人类（例如AI需要完整程序才能分辨"香"和"臭"）
- 如果人类介入AI的数据处理过程，弥补"最后一公里"的不足
- 数字生命与AI结合 → **快速迭代出超级人工智能**

#### 课题：数字生命的实现方式

**科学顾问建议的技术路径**：
1. 参照当前**脑机接口**方式——图丫丫头部佩戴脑电极帽子
2. 图恒宇的"数字生命卡"既要像U盘可插拔，还要有计算功能
3. 数字生命卡的结构、形状、体积、重量经科学顾问详细设计

#### 课题：图丫丫记忆3万位密码是否合理？

- 科学顾问提供了**多组密码方案**（二进制、十进制、三十六进制）
- 最终选择了**十进制+逐个输入**的视觉呈现效果
- 现实中使用的密码远不止3万位，而且要复杂得多

#### 剧中AI细节的科学支持
- 屏幕上每一行代码——科学顾问一行一行推敲出来的
- 自动发送系统邮件中的命令行命令和反馈结果——经科学顾问设计
- 刘德华饰演的图恒宇**打字编程的手势**——科学顾问拍摄了自己编程打字的录像供演员模仿

### 2.3 材料/工程组

#### 课题：防水喷雾
- 科学顾问咨询了材料专家
- 提供了两个方案：**生物防水技术**（依靠细菌修复）和**高分子复合材料技术**
- 生物防水技术用于剧情设定
- 高分子复合材料技术类似现有的"超滑马桶"原理，用于地球停转后防止海水倒灌

#### 课题：太空电梯隔热
- 方案一：蜂窝状隔热瓦，烧蚀降温
- 方案二：**通过水与冰的相变完成降温**（电影最终选用的方案）
- 致敬经典科幻小说中的隔热方式

#### 课题：太空电梯训练营考题
- 由中科院力学所的科学顾问亲自设计

### 2.4 天文/物理组

#### 课题：地球停转后的时间变化
- 科学家通过测算：地球停止自转后，一天=**60个小时**
- 这意味着时钟显示的时间可能是"35点20分"
- 整个历法体系随之改变

#### 课题：现实中的"流浪地球"
- 苟利军介绍：人类已发现**近30颗**正在流浪的行星
- 它们处于非常黑暗的区域，无法反射恒星光芒
- 完美隐蔽在宇宙空间中

## 三、科学顾问工作方法论

### 3.1 工作流程

```
前期开发 → 世界观搭建 → 剧本讨论 → 科学合理性评估
    ↓           ↓              ↓              ↓
科学文本 → 分章节设定 → 逐场修改 → 视觉呈现调整
    ↓           ↓              ↓              ↓
 拍摄阶段 → 道具指导 → 演员动作指导 → 后期确认
```

### 3.2 核心原则

1. **科幻不等于科学**：科幻有科学的基础，但一定有浪漫的想象
2. **视觉优先**：在科学与视觉效果之间做取舍
3. **平衡原则**：既要有"科学合理"，也要有"视觉艺术"
4. **提前介入**：科学顾问在开拍前就参与，避免拍摄后再修改

### 3.3 科学顾问的约束

- 与片方签订保密协议，直到电影上映才知道哪些想法被采用
- 有些改动"牵一发而动全身"，一个科学建议可能导致整个剧情线调整
- 最终呈现要在6-7条假设同时成立的前提下才能实现

### 3.4 剧本修改数据

> 根据崔原豪博士所述：**约三分之一的剧本内容经过科学顾问调整**
