# 流浪地球官方世界观与完整时间线

> **整理说明**：以下内容综合自《流浪地球》电影制作手记中的官方编年史、媒体公开资料、科学顾问访谈。时间线基于官方设定（1977年—2075年），涵盖两部电影及编年史主要事件。

## 一、世界观核心框架

### 1.1 太阳危机背景

> 太阳核心氢转氦速度异常加速，预计**400年内**将发生致命**氦闪**，太阳死亡膨胀后将吞噬地球轨道。

- 提出者：中国天文学家**付建明**（1977年首次提出论点）
- 核心困境：科学界长期质疑 → 2026年太阳风暴证实 → 全球紧急应对

### 1.2 流浪地球计划

"流浪地球"计划 = 在地球表面建造**12000台**行星发动机，推动地球脱离太阳系，前往**4.2光年**外的比邻星系，总航程约**2500年**，需要**100代人**接力完成。

**计划四阶段**：

| 阶段 | 时间 | 内容 |
|------|------|------|
| 刹车时代 | 2039—2042年 | 行星发动机启动，地球停止自转 |
| 逃逸时代 | 2042年以后 | 地球加速至逃逸速度，脱离太阳系 |
| 流浪时代Ⅰ（加速） | — | 持续加速飞往比邻星 |
| 流浪时代Ⅱ（减速） | — | 接近比邻星时掉头减速 |
| 新太阳时代 | — | 进入比邻星轨道 |

### 1.3 联合地球政府（UEG）

- 各国向联合国释出部分军权，组建联合地球政府
- 下设：联合政府大会、地球安全军（ECF）、行星发动机控制中心
- **三大应对计划**：
  - **移山计划**：验证并推动行星发动机建造（最终执行方案）
  - **方舟计划**：建造领航员空间站，作为备份方案
  - **逐月计划**：放逐月球，避免地月相撞

### 1.4 地下城

- 深度：据设定约**500米**（小说）至**5000米**（电影北京地下城）
- 容纳人口：约**35亿人**（全球总人口约半数）
- 抽签机制决定入城资格
- 岩浆渗入是地下城最大威胁

### 1.5 行星发动机

- **高度**：约11000米（超过珠穆朗玛峰）
- **原理**：重聚变技术（将岩石作为燃料）
- **分布**：亚洲和美洲大陆的完整坚实板块上
- **数量**：12000台（其中有些分布于地球海洋）

### 1.6 数字生命计划

- 数字生命：将人类意识上传至量子计算机
- 核心争议：伦理问题、法律地位、资源分配
- 被联合政府关停后，演变为**数字生命派**反叛运动
- 代表人物：图恒宇（秘密推进）、马兆（550系列研发者）

### 1.7 领航员计划

- 领航员空间站作为地球流浪的导航、通信、监测中心
- 人类航天员在太空中执行长期任务
- 电影中MOSS是空间站核心AI

## 三、世界观细节设定

- **社会分层**：地下城中签机制导致的人口分化
- **文化教育**：地下城学校教育、黄金时代回忆

- **天文历法**：地球停转后一天=**60小时**（时钟显示可能出现"35:20"等）

- **生物工程**：人造生物圈技术、地下农业体系

## 四、关键科学概念

### 氦闪
太阳核心氦聚变失控爆发，释放能量将摧毁地球生态。

### 重聚变
超越现有核聚变概念，以重元素（如硅、岩石）为燃料的高能核反应。

### 相控阵
通过控制阵列中每个单元的能量释放时序，使能量在特定方向聚焦叠加。

### 人在回路（Human-in-the-loop）
人类介入AI决策过程，弥补AI在"最后一公里"的不足，数字生命与AI结合产生超级人工智能。
