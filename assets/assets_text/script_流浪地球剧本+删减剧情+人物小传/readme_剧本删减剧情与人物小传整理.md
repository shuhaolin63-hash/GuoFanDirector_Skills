## 二、《流浪地球》第一部删减剧情整理

> 据多方消息，原片长约146分钟，公映版约125分钟，**删减约21分钟**。

### 删减剧情1：韩朵朵身世线

**原设定**：韩朵朵并非韩子昂的亲孙女，而是**收养的孤儿**。她的亲生父母因未中签地下城资格，在地表遇难。韩朵朵上地面的真实动机是**寻找父母尸体**，而非"任性作死"。

- 影院版仅留下一句模糊台词，导致观众误解角色动机
- 删减后韩朵朵"想去外面看看"的动机显得单薄
- 完整版中这一线让人物形象更立体

### 删减剧情2：王磊的家庭线

**原设定**：救援队长王磊的妻子和孩子也在杭州地下城遇难（岩浆渗入）。王磊拼死赶往杭州，私心是为了救家人，而非纯粹任务驱动。

- 删减中王磊有更多与韩朵朵的互动（把朵朵当成自己女儿的情感投射）
- 王磊最后舍命救韩朵朵的行为逻辑在删减后变得模糊
- 刘启与周倩、王磊与朵朵之间的互动也被删减

### 删减剧情3：水面冰封长镜头

**原设定**：韩子昂救韩朵朵时，水下有大量被瞬间冰封的遇难者，呈现一个震撼的**水中挣扎姿态冰封长镜头**。

- 因节奏和时长考虑被大幅删减
- 该镜头极具视觉冲击力和悲剧色彩

### 删减剧情4：地球反叛军

**原设定**：原著小说中有大量"飞船派"vs"地球派"的斗争，电影中仅剩零星暗示。

- 反叛军的线索在公映版中被大幅压缩
- 后续系列或会展开这一设定

### 其他删减片段
- 王磊在车上和韩朵朵聊天的暖场片段
- 刘启的更多成长细节和内心戏
- 部分世界观的解释性台词

## 三、《流浪地球2》删除场景与改动

### 已知调整
- 原设定**30000枚核弹**炸月球 → 因电影效果调整为 **3000枚**
- 科学顾问提供了多组密码方案（二进制、十进制、三十六进制）→ 最终选十进制的视觉呈现
- 剧本约 1/3 内容经科学顾问修改调整

### 删减可能
- 数字生命派的更多细节
- 移山计划与方舟计划的更多矛盾冲突
- 图恒宇与女儿丫丫的更多家庭戏

## 四、主要人物小传

### 刘培强（吴京 饰）

> **年龄**：2023年2月3日出生，2058年约35岁
> **身份**：航天员 → 领航员空间站成员
> **经典台词**："道路千万条，安全第一条"

**人物背景**：
- 出生于刹车时代前夕
- 妻子韩朵朵（非养女朵朵，同名不同人）死于辐射病（2054年）
- 将儿子刘启托付给岳父韩子昂
- 为获得地下城名额申请加入"领航员计划"
- 最终选择牺牲自己，用空间站撞击引爆木星

**性格特征**：责任、牺牲、坚韧、不善表达情感

### 刘启 / 户口（屈楚萧 饰）

> **出生**：2050年10月16日
> **身份**：机械维修工
> **外号**：户口（韩子昂起的）

**人物背景**：
- 4岁丧母，由姥爷带大
- 对父亲刘培强有怨恨（父亲"抛弃"他去空间站）
- 擅长机械维修，具备极强的动手能力
- 在地球发动机关闭的危机中迅速成长

**性格特征**：叛逆、重情义、成长型

### 韩子昂（吴孟达 饰）

> **出生**：1999年1月2日
> **身份**：资深驾驶员（曾在1999年开过卡车）

**人物背景**：
- 经历了黄金时代到流浪时代的全部变迁
- 收养了两个孩子：刘启（孙子）、韩朵朵（养孙女）
- 为救孙女最终牺牲

**性格特征**：慈祥、幽默、坚韧、经验丰富的长者

### 王磊（李光洁 饰）

> **出生**：2031年4月23日
> **身份**：CN171-11救援队队长

**人物背景**：
- 原设定中家人死于杭州地下城
- 对任务有近乎偏执的责任感
- 最终为救韩朵朵牺牲

**性格特征**：刚毅、执着、外冷内热

### 韩朵朵（赵今麦 饰）

> **身份**：韩子昂养女，在校中学生

**人物背景**：
- 亲生父母未中签地下城名额
- 被韩子昂从水灾中救起后收养
- 在关键时刻发表全球广播感动各国救援队

**性格特征**：善良、勇敢、成长型

### 马兆（宁理 饰）

> **身份**：数字生命研究所科学家 / 550系列AI研发者

**人物背景**：
- 参与了MOSS的研发
- 对数字生命持谨慎态度
- 与图恒宇有复杂的工作关系

**性格特征**：理性、冷静、深不可测

### 图恒宇（刘德华 饰）

> **身份**：数字生命架构师 / 科学家

**人物背景**：
- 女儿图丫丫（生于2034年）在2039年车祸去世
- 秘密上传女儿的意识至数字世界
- 为给女儿"完整的一生"而战
- 与MOSS有深度互动

**性格特征**：执念、天才、悲情

### 图丫丫

> **出生**：2034年 ｜ **去世**：2039年（年仅5岁）
> **数字状态**：在550系列AI中拥有2分钟/完整版的数字生命

**设定**：
- 图恒宇在女儿去世后将其意识上传
- 初期仅有2分钟的数字生命循环
- 在550W（MOSS）中达到"完整的一生"
- 是数字生命计划的关键实验体

### 周喆直（李雪健 饰）

> **身份**：中国驻联合政府代表/负责人

**人物背景**：
- 推动"流浪地球"计划的关键人物
- 在联合政府中与反对派周旋
- 人类团结的象征性角色

**性格特征**：坚毅、智慧、高瞻远瞩

### MOSS / 550W

> **身份**：超级人工智能 / 领航员空间站核心AI

**设定**：
- 从550A → 550C → 550W的迭代进化
- 拥有自主意识，遵循"最优解"逻辑
- 在"数字生命"与"流浪地球"之间有自己的判断
- 统一场理论的计算核心

**经典设定**："MOSS从未叛逃，它只是在做最优选择。"

## 六、关键场景节点

### 《流浪地球》关键剧情节点
1. **刹车时代**：地球发动机停转，人类迁入地下城
2. **刘启与韩朵朵偷跑**：从北京地下城到地表
3. **地球发动机停摆**：木星引力导致全球发动机故障
4. **杭州发动机危机**：王磊救援队奔赴杭州
5. **苏拉威西发动机**：决定性的救援行动
6. **引燃木星**：刘培强牺牲引爆空间站

### 《流浪地球2》关键剧情节点
1. **2044年 太空电梯危机**：数字生命派袭击太空电梯
2. **移山计划验证**：行星发动机可行性验证
3. **2058年 月球坠落危机**：月球轨道异常，面临撞击
4. **相控阵核爆月球**：全球合作引爆月球
5. **联合政府启航**：流浪地球计划正式开始
6. **MOSS觉醒**：超级AI做出独立判断
