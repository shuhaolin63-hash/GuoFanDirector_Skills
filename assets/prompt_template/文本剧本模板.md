# 文本剧本模板

> 短剧剧本标准输出模板 —— 适用于郭帆导演工作室短剧项目

---

## 1. 文件头元数据

```
---
剧本名称:       《》
集数:           第 N 集
总字数:         约 XXXX 字
题材:           [甜宠/逆袭/战神/悬疑/都市/古装/穿越]
编剧:           
版本:           V1.0
创建日期:       YYYY-MM-DD
---
```

---

## 2. 分集标题与信息

| 字段 | 内容 |
|------|------|
| **分集标题** | 每集一个核心标题，字数 ≤ 8 字，直击本集核心冲突 |
| **集字数** | 800-1200 字（标准短剧单集） |
| **场景数** | 3-5 场 / 集 |
| **出场人物** | 本集出场人物列表，标注核心/次要 |
| **本集核心钩子** | 结尾处悬念/反转一句话（用于剪辑卡点） |

---

## 3. 分场格式

每场由以下元素构成：

### 场标

```
【第 N 场】场景名称 | 时间 | 人物
```

- **场号**：连续数字编号，01/02/03...
- **场景**：精确到具体空间（如"总裁办公室-落地窗前"而非"办公室"）
- **时间**：晨/午/暮/夜（如需更精确标注具体时刻）
- **人物**：该场次出场人物（核心人物在前）

### 动作描写

```
【动作】描述内容
```

- 动作描写必须具象化、细节写实、可拍摄
- 禁止抽象形容词（如"悲伤地"）→ 替换为可拍摄细节（"眼眶泛红，手指微微颤抖"）
- 每个动作描写 ≤ 30 字，一句话一个动作
- 每个动作段落 ≤ 3 句，之后切台词或换行

```
【正确示例】
【动作】林深右手食指在红酒杯沿上缓缓划过半圈，发出细微的摩擦声。
【动作】苏晚脚步在门口顿住——手还搭在门把上，没有回头。

【错误示例】
【动作】林深很生气。
【动作】苏晚犹豫了一下。
```

### 台词格式

```
人物名：台词内容（<=15字）
```

- **单句台词 ≤ 15 字**（含标点）
- 超过 15 字必须拆分，中间用动作打断
- 停顿标注：使用 **/** 表示 0.5 秒停顿，**//** 表示 1 秒以上停顿
- 语气词使用实际发音（"嗯"、"啊"、"哈"），禁止书面语化台词

```
【正确示例】
林深：你以为//就你委屈？（手指松开领带结）
苏晚：嗯。
林深：说话。
苏晚：我没什么好说的了。

【错误示例】
林深：你以为这个世界上只有你一个人觉得委屈吗？
苏晚：我确实没有什么想要说的了。
```

### 情感标注

情感标注仅限可拍摄（可摄影）的具象化微表情描述，**不得使用抽象情感词**。

```
标注格式：在台词或动作后以括号标注
```

| 可标注项（具象） | 禁止标注项（抽象） |
|------|------|
| （嘴角微抽） | （生气） |
| （瞳孔微缩） | （震惊） |
| （喉结滚动） | （紧张） |
| （指尖陷进掌心） | （愤怒） |
| （呼吸顿了一拍） | （害怕） |
| （睫毛轻颤） | （伤心） |
| （指节捏得泛白） | （压抑） |

### 节奏标注

节奏标注用 `【节奏】` 标记，每 15-20 秒必须有一个节奏变化点。

```
标注格式：【节奏：加速/减速/骤停/反转/悬念】
```

- **加速**：台词对切加快、动作密度增加 → 用于情绪攀升
- **减速**：静默、慢镜头、长停顿 → 用于情绪沉淀或反转前蓄力
- **骤停**：突然的动作静止或台词中断 → 用于惊愕/发现
- **反转**：立场/情绪/情境瞬间翻转 → 用于爽点/痛点
- **悬念**：话说到一半、发现某物、听到某声 → 用于结尾钩子

**每集至少包含：**
- 开头 3-5 秒一个节奏加速（钩子钩住）
- 中间 2-3 个节奏波动点
- 结尾悬念留白（画面定格 + 音效延续）

## 4. 完整示例场次

```
---
剧本名称：《替身前妻她不哭了》
集数：第 1 集
总字数：约 1050 字
题材：甜宠 / 逆袭
编剧：郭帆工作室
版本：V1.0
创建日期：2026-07-20
---

【分集标题】离婚协议

【场景数】4场
【出场人物】苏晚（核心）、林深（核心）、助理（次要）

---

【第 01 场】林氏总裁办公室-落地窗前 | 夜 | 林深、苏晚

【节奏：骤停】
【动作】苏晚把离婚协议轻轻放在桌面上。
【动作】林深背对着她，右手食指在红酒杯沿上缓缓划了半圈，发出细微的摩擦声。

林深：（头也不回）你要的，我都给你。

【动作】苏晚的手指在纸张边缘捏紧了一瞬，又松开。

苏晚：嗯。

【节奏：减速】
【动作】林深转过身来，目光从她脸上扫过，落在协议上。

林深：不后悔？

【动作】苏晚嘴角弯了一下——弧度刚好够他看见，又刚好够说是礼貌。

苏晚：林总开出的条件//很好。

【节奏：加速】
林深：（走近两步）条件？（轻笑）你以为我在谈生意？

苏晚：不是吗？

林深：苏晚，我们结婚三年，你一直很聪明——（停顿）聪明到让我讨厌。

【动作】苏晚抬起眼，视线与他平齐。眼底没有波澜。

苏晚：那正好，我也讨厌你。

【节奏：骤停】
【动作】林深捏着酒杯的手猛地收紧——指节泛白，酒液晃动了一下。

---

【第 02 场】林氏大厦一楼大厅 | 夜 | 苏晚

【节奏：加速】
【动作】苏晚踩着高跟鞋快步走出电梯。
【动作】她一手攥着手包，一手抹了一下眼角——没有眼泪，只有一点洇开的睫毛膏。
【动作】门口正在下雨。

苏晚：（低声//自嘲）连老天爷都会挑时候。

【动作】她站在大厅门口，看着外面的雨幕，深吸一口气。
【动作】手机震动——来电显示：陌生号码。
【动作】她按下接听。

对方（OS）：苏小姐，您爷爷留下的那块地，有人出价三个亿。

【节奏：悬念】
【动作】苏晚瞳孔微缩，握手机的手指收紧。

苏晚：谁？

（画面定格，音效——一声沉闷的雷声）

【第 02 场完】
```
