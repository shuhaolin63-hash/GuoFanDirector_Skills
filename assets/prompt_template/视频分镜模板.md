# 视频分镜模板

> 竖屏短剧分镜表格模板 —— 适用于郭帆导演工作室短剧项目

---

## 1. 分镜表格结构

| 镜头号 | 景别 | 运镜 | 画面描述 | 台词 | 时长 | 备注 |
|--------|------|------|----------|------|------|------|
| 连续编号 | 详见景别定义 | 详见运镜说明 | 具象画面内容 | 本镜头内台词 | 秒数 | 色调/特效/剪辑参考 |

### 列规则

| 列名 | 规则 |
|------|------|
| **镜头号** | 三位连续编号，如 001/002/003... 每集从 001 重新开始 |
| **景别** | 遵循下文景别编码 |
| **运镜** | 遵循下文运镜编码 |
| **画面描述** | 具象化视觉描述，一句话 ≤ 25 字，可拍摄。标注竖屏构图信息（人脸位置/核心动作区域） |
| **台词** | 遵循台词规范（单句 ≤ 15 字），无台词标注"（无台词）"或"（OS：画外音）" |
| **时长** | 每个镜头 2-5 秒，关键镜头可达 6-8 秒，单位秒，整数 |
| **备注** | 色调绑定（引用色调模板编号）、特效标注、剪辑参考、情绪提示 |

## 2. 景别定义与画框范围

| 景别编码 | 名称 | 定义与画框范围 |
|----------|------|---------------|
| **XJ** | 极限特写 | 仅限眼部（单只眼）或嘴唇（唇部特写）或手部细节（指尖/戒指/伤口）。画框范围 < 人脸面部的 1/4。传达极致情绪或关键道具信息。 |
| **TX** | 特写 | 眼/脸/唇/手/道具细节。画框范围：面部（额头到下巴）或单手或单个道具。 |
| **JT** | 近景 | 胸部以上。核心叙事景别。人物面部表情 + 上半身微动清晰可见。**竖屏主景别（占比 ≥ 60%）** |
| **ZJ** | 中景 | 腰部以上或全身的 1/2。多用于室内两人对话、人物与环境关系的交代。 |
| **QJ** | 全景 | 全身 + 环境。交代空间关系、人物入场/退场。每集 ≤ 2 个全景为宜。 |
| **YT** | 远景 | 大环境 + 小人影。用于转场、时间推移、情绪拉开。每集 ≤ 1 个。 |

### 竖屏景别约束

| 约束项 | 要求 |
|--------|------|
| **近期比例** | 近景（JT）≥ 60%，特写（TX+XJ）20-30%，中景（ZJ）10-15%，全景（QJ）+远景（YT）≤ 5% |
| **竖屏人脸位置** | 核心人脸位于画面上 1/3 区域，头顶留白 ≤ 1/5 画面高度 |
| **核心动作居中** | 核心动作（对话/眼神/手部动作）位于画面中线两侧范围内 |
| **上下留白比例** | 上留白 : 下留白 = 1 : 2（竖屏阅读习惯，重心偏下） |
| **多人物构图** | 双人对话采用上下分区构图（非左右），主人物在上/前景，次人物在下/后景 |

## 3. 运镜说明

| 运镜编码 | 名称 | 用法说明 | 典型场景 |
|----------|------|----------|----------|
| **G** | 固定镜头 | 摄像机静止。用于定格反转、情绪释放瞬间、台词对切。**基础镜头，占比 ≥ 50%** |
| **T** | 推镜 | 镜头匀速推进。用于聚焦情绪（从近景推到特写）、揭示关键道具。推进速度：普通 2-3 秒，情绪慢推 4-5 秒。 |
| **L** | 拉镜 | 镜头匀速拉远。用于情绪拉开、场景揭示、转场。 |
| **Y** | 移镜 | 水平或垂直移动。用于跟随叙事（跟随人物转身/走路）。垂直移镜适用于竖屏（上下摇移）。 |
| **G-T** | 固转推 | 先固定 1-2 秒，再慢推。用于情绪积累到爆发。 |
| **G-L** | 固转拉 | 先固定 1-2 秒，再慢拉。用于反转或真相揭示。 |
| **MT** | 慢推 | 极慢速推进（5-8 秒）。用于制造压迫感/悬疑感。呼吸感推进。 |
| **YP** | 摇拍 | 水平摇移。用于环境扫描、人物反应对比。 |
| **SD** | 手抖 | 手持拍摄抖动效果。用于紧张/慌乱/追逐场景。 |

### 运镜配比规则

| 运镜类型 | 占比建议 |
|----------|----------|
| 固定镜头（G） | ≥ 50% |
| 推拉镜头（T/L/G-T/G-L） | 20-25% |
| 移镜/摇拍（Y/YP） | 10-15% |
| 特殊运镜（MT/SD） | ≤ 10% |

## 5. 完整示例（一个短剧片段的分镜）

```
---
项目名称：《替身前妻她不哭了》
集数：第 1 集 - 离婚协议
片段：开场至分别
总时长：约 35 秒
总镜头数：10 镜
色调模板：冷峻都市（模板 A）
---

| 镜头号 | 景别 | 运镜 | 画面描述 | 台词 | 时长 | 备注 |
|--------|------|------|----------|------|------|------|
| 001 | ZJ | G | 林深背对镜头站在落地窗前，窗外夜景霓虹。桌上一份文件。 | （无台词） | 3s | 色调A；环境音：空调低鸣 |
| 002 | TX | G-T | 镜头固定2s后慢推到文件封面上"离婚协议书"五个字。 | （无台词） | 4s | 色调A；推镜从2s开始 |
| 003 | JT | G | 苏晚正面入画，站在桌前。手捏着包带，指节微白。 | 苏晚：我签好了。 | 3s | 色调A；人脸位于画面上1/3 |
| 004 | TX | T | 从苏晚面部慢推到她的眼睛——睫毛轻颤，强忍泪水。 | （无台词） | 3s | 色调A；竖屏单眼位于中央 |
| 005 | JT | G | 林深转身，视线落在她脸上。表情冷硬。 | 林深：不后悔？ | 2s | 色调A；上下双人构图 |
| 006 | TX | G | 苏晚嘴角微微弯了一下——肌肉在牵动。 | 苏晚：嗯。 | 2s | 色调A；唇部特写 |
| 007 | JT | G | 林深走近两步，身体挡住部分灯光。 | 林深：很好。（停顿）那就这样吧。 | 3s | 色调A；暗部增加反差 |
| 008 | XJ | MT | 慢推到苏晚拿着笔的手指——指腹在笔杆上来回摩擦。 | （无台词） | 4s | 色调A；压迫感慢推5s |
| 009 | QJ | L | 从苏晚单人中景缓慢拉远为全景，她独自站在办公室中央。 | （无台词） | 4s | 色调A；拉镜+情绪留白；转场准备 |
| 010 | TX | G | 文件封面上签名特写——墨迹未干，最后一笔拖长了。 | （OS：雷声） | 3s | 色调A；结尾悬念；字幕叠"第1集完" |
```

### 分镜核查清单

每条分镜完成后需核对以下内容：

- [ ] 每个镜头画面描述是否可拍摄（避免抽象描述）
- [ ] 景别分配是否符合竖屏规则（JT ≥ 60%）
- [ ] 运镜变化是否服务于叙事目标（避免为炫技而运镜）
- [ ] 台词是否 ≤ 15 字（含标点）
- [ ] 色调标注是否已绑定三套模板之一
- [ ] 总时长是否在目标区间内（单集 60-90 秒）
- [ ] 结尾镜头是否留悬念/留白
- [ ] 竖屏构图中人脸是否在上 1/3 区域
