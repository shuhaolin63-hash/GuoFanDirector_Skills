## 3. 节奏标注格式

### 标注符号

| 符号 | 含义 | 使用场景 |
|------|------|----------|
| `|` | 短停顿（0.3-0.5 秒） | 断句、自然呼吸 |
| `||` | 中停顿（0.8-1.2 秒） | 情绪转折、悬念蓄力 |
| `|||` | 长停顿（1.5-2.5 秒） | 节奏骤停、反转前留白 |
| `>` | 加速（语速提升 20%） | 情绪攀爬、争执对切 |
| `<` | 减速（语速降低 20%） | 情绪沉淀、回忆、沉思 |
| `^` | 重音 | 强调关键词，"我^不^后悔" |
| `~` | 拖音 | 尾音延长，"你~说~呢" |
| `(b)` | 呼吸声 | 吸气/叹息/抽气，增加沉浸感 |

### 台词节奏标注格式

```
人物名：^台词内容^ | ~尾字拖长~ || 下句内容 (b)
```

**示例：**

```
林深：（低沉）你以为||我^在^乎^吗？(b)
苏晚：你~说~呢？
林深：>好>很好>非常好 || 那^就这样吧。(b)
```

每个人物在项目开头定义一个声音标签卡，用于全剧一致性控制。

```

---

音色：清亮中高音，齿音清晰
语速基准：150 字/分钟
音域：C3-G4
气声比例：30%（悲伤场景升到 50%）
共鸣位置：口腔前部
口音：标准普通话（无地方口音）
咬字：清晰，字尾干净
---
声音特征标签：柔亮、韧性、微气声、尾音上扬（疑问时）、齿音清晰
情绪色标：
  - 平静：#A0C4E8（淡蓝）
  - 委屈：#C9A0C4（淡紫）
  - 坚定：#E8A0A0（淡红）
  - 悲伤：#A0A0A0（灰色）
---
```

### 声音标签字段说明

| 字段 | 说明 |
|------|------|
| **人物** | 角色名称 |
| **音色** | 主观音色描述 + 客观特征 |
| **语速基准** | 正常叙事语速（字/分钟） |
| **音域** | 推荐音域范围 |
| **气声比例** | 正常状态与特殊状态的气声占比 |
| **共鸣位置** | 口腔/鼻腔/胸腔 |
| **口音** | 标准/方言/口音特征 |
| **咬字** | 清晰度与字尾处理方式 |
| **声音特征标签** | 3-6 个关键词标签 |
| **情绪色标** | 核心情绪对应的听觉"色彩"（用于混音参考） |

## 5. 音效标注格式

### 音效标注规则

音效在剧本中标注在 `【音效】` 标记内，格式如下：

```
【音效：类型 - 描述 (时长)】
```

### 音效分类

| 类型编码 | 名称 | 示例 | 用法 |
|----------|------|------|------|
| **AMB** | 环境音 | 空调低鸣/雨声/街道嘈杂/鸟鸣 | 贯穿全场景，确立空间感 |
| **SFX** | 特效音 | 脚步声/关门声/玻璃破碎/手机震动 | 与动作一一对应，精准匹配画面 |
| **EMO** | 情绪音效 | 心跳声/一声轻叹/尖锐耳鸣 | 渲染情绪，通常配合特写镜头 |
| **TRN** | 转声音效 | 风声呼啸/时钟滴答快速/布料抽动 | 用于场景转换或时间流逝 |
| **IMP** | 冲击音效 | 耳光/拳击/重物落地/爆炸 | 强烈感官刺激，配合节奏骤停 |

### 音效标注示例

```
【音效：AMB - 咖啡机低鸣 + 杯碟碰撞声 (贯穿全场)】
【音效：SFX - 高跟鞋敲击大理石地面，由远及近 (3s)】
【音效：EMO - 一声沉重的心跳 (0.5s，配合瞳孔收缩画面)】
【音效：TRN - 风声呼啸 + 画面淡出 (2s，场次转场)】
【音效：IMP - 酒杯碎裂声 (0.3s，节奏骤停点)】
```

## 6. 配乐标注格式

### 配乐标注规则

配乐在剧本中标注在 `【配乐】` 标记内，格式如下：

```
【配乐：选曲风格 | 情绪 | 起止点 | 音量百分比】
```

### 配乐要素

| 要素 | 格式 | 示例 |
|------|------|------|
| **选曲风格** | 具体风格名称 | 钢琴独奏/弦乐四重奏/电子低音/Guitar Fingerstyle |
| **情绪** | 情绪关键词 | 压抑/紧张/温暖/悲壮/悬疑 |
| **BPM** | 每分钟拍数 | BPM 60 / BPM 120 |
| **起止点** | 场次编号或时间码 | 从第 03 场开始 → 第 05 场结束 / 00:15-00:45 |
| **音量百分比** | 与台词的比例 | 30%（台词下垫底）/ 60%（纯音乐段落）/ 0%（对话区） |

### 配乐标注示例

```
【配乐：弦乐四重奏 | 压抑+悬疑 | BPM 65 | 第01场全程 | 音量40%】
【配乐：钢琴独奏（高音区单音）| 悲伤 | BPM 50 | 00:15-00:35 配合苏晚特写 | 音量50%】
【配乐：电子低音渐入 | 紧张攀升 | BPM 110 | 第04场中段开始 → 片尾 | 音量30%→70%渐强】
【配乐：静音 | — | — | 台词对切密集区 | 音量0%（无配乐）】
```

### 配乐起止规则

| 场景类型 | 配乐规则 |
|----------|----------|
| 对话密集场 | 配乐 ≤ 30% 音量，弦乐/钢琴低音区垫底，不影响台词 |
| 纯情绪场（无台词） | 配乐 60-80%，主旋律出场 |
| 动作追逐场 | 节奏配乐 BPM ≥ 120，音量 50-70% |
| 结尾悬念 | 配乐渐强至 70% 后骤停 → 留 1-2 秒静默 → 片尾音效 |
| 转场 | 配乐延续 1-2 秒跨场（声音叠化桥接） |
