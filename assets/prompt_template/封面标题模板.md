## 2. 标题结构

### 标准标题公式

```
[悬念钩子] + [信息点] + [留白]
```

| 要素 | 说明 | 字数限制 | 示例 |
|------|------|----------|------|
| **悬念钩子** | 引发好奇/冲突/疑问 | 4-8 字 | "离婚协议签完"、"她竟转身就走" |
| **信息点** | 核心剧情提示 | 4-10 字 | "三亿遗产从天而降"、"总裁追悔莫及" |
| **留白** | 省略号或开放式结尾 | 不必须 | "……"、"？！" |

### 标题总字数限制

| 平台 | 标题最大字数 | 最佳字数 |
|------|-------------|----------|
| 抖音 | ≤ 25 字 | 12-18 字 |
| 快手 | ≤ 20 字 | 10-15 字 |
| 视频号 | ≤ 25 字 | 12-18 字 |
| 小红书 | ≤ 20 字 | 10-15 字 |

### 单集标题要求

- **主标题**（封面大字）：≤ 8 字，加粗，直击本集核心冲突
- **副标题**（封面小字/描述）：≤ 20 字，补充剧情钩子
- **系列标题**（整剧标题）：≤ 10 字，品牌统一

## 3. 按题材分类的标题词库

### 甜宠题材

| 类型 | 句式示例 |
|------|----------|
| 追妻火葬场 | "离婚后他疯了"、"总裁跪求复合"、"悔不当初的他" |
| 契约婚姻 | "协议老公动真心"、"假结婚真动情"、"太太你别跑" |
| 先婚后爱 | "闪婚老公是千亿总裁"、"结婚三年他不认识我" |
| 霸道宠溺 | "他把我宠上天"、"全城都知道他爱我" |

**标题句式模板：**
- [动作] + [人物] + [结果]："离婚协议一签，他后悔了"
- [反问/疑问] + [信息]："谁说她配不上他？"
- [对比] + [反转]："被扫地出门那天，他跪了"
- [具体场景] + [情绪词]："雨夜，他第一次为我撑伞"

### 逆袭/战神题材

| 类型 | 句式示例 |
|------|----------|
| 身份反转 | "外卖小哥是隐藏首富"、"保安竟是退役战神" |
| 打脸爽文 | "全公司等着看她笑话"、"下一秒全场安静" |
| 复仇回归 | "三年后她强势归来"、"当年跪着求他的人" |

**标题句式模板：**
- [身份反差] + [动作]："乡下舅妈竟是国宴大厨"
- [等待打脸] + [反转]："都在等她出丑，直到她亮出身份"
- [时间] + [归来]："三年后，她带着千亿资产回来了"

### 悬疑题材

| 类型 | 句式示例 |
|------|----------|
| 反转悬疑 | "杀死我的人竟然是他"、"真相反转再反转" |
| 细思极恐 | "我家的保姆从来不笑"、"镜子里的我在对我笑" |
| 涉案剧情 | "第13号目击者"、"那个雨夜发生了什么" |

**标题句式模板：**
- [反常现象] + [疑问]："床头的神秘纸条是谁放的？"
- [否定+真相]："没有人相信我，但我真的看见了"
- [时间节点] + [事件]："凌晨三点，她敲了我的门"

### 古装/穿越题材

| 类型 | 句式示例 |
|------|----------|
| 穿越逆袭 | "穿成废柴后我爆红全京城"、"史上最强皇后是她" |
| 宫斗权谋 | "她一开口，全场鸦雀无声"、"王爷跪求她原谅" |

## 4. 封面设计参数

### 设计参数总表

| 参数 | 规格 |
|------|------|
| 画幅 | 9:16 竖屏 |
| 分辨率 | 1080x1920 px（最低） |
| 文件格式 | JPG/PNG，RGB 色彩空间 |
| 文件大小 | ≤ 5MB |

### 人脸位置

| 人物数量 | 人脸位置 | 占比区域 |
|----------|----------|----------|
| 单人（女性） | 画面中心偏左/偏右 20%，自上 1/4 处 | 画面上 1/3 |
| 单人（男性） | 画面中心偏左/偏右 20%，自上 1/4 处 | 画面上 1/3 |
| 双人 | 主人物占画面 60% + 次人物占 30%（上下或前后分区） | 总人脸区 ≤ 画面 2/3 |
| 多人 | 核心人物居中放大，其余虚化或侧面 | 核心人脸 ≥ 画面 1/3 |

### 文字位置

| 文字层级 | 位置 | 字体大小 | 样式 |
|----------|------|----------|------|
| **主标题** | 画面中下部（黄金分割线以下） | 72-96pt | 加粗，字体醒目，全角字符 |
| **副标题/标签** | 主标题上方或下方留白区 | 36-48pt | 细体/常规 |
| **角标/系列名** | 左上角或右上角 | 24-30pt | 统一品牌字体 |
| **悬疑提示词** | 画面底部或人脸侧方 | 48-60pt | 字体可艺术化处理 |

### 文字避让规则

- 文字区域避开人脸面部（尤其是眼睛区域）
- 标题文字下边缘距画面底部 ≥ 150px
- 标签文字与主标题间距 ≥ 30px
- 文字与背景之间必须有足够的对比度（对比度 ≥ 4.5:1）

### 色彩方案

| 题材 | 主色调 | 辅色调 | 文字颜色 | 色调倾向 |
|------|--------|--------|----------|----------|
| 甜宠 | #FFB6C1（浅粉） | #FFD700（金色） | 白色/深灰 | 暖调柔和 |
| 逆袭 | #1a1a2e（深蓝黑） | #e94560（红） | 白色/金色 | 冷峻高对比 |
| 战神 | #0f0f0f（纯黑） | #C0C0C0（银灰） | 白色/银色 | 暗调金属感 |
| 悬疑 | #2d3436（暗灰绿） | #d63031（暗红） | 白色/暗红色 | 冷暗低饱和 |
| 古装 | #8B4513（棕褐） | #DAA520（金黄） | 金色/白色 | 暖调复古 |
| 都市 | #2c3e50（深蓝灰） | #ecf0f1（白） | 白色 | 现代简洁 |

### 字体风格

| 位置 | 推荐字体 | 样式 |
|------|----------|------|
| 主标题 | 思源黑体 Bold / 阿里巴巴普惠体 Bold | 加粗，行间距 1.2 |
| 副标题 | 思源黑体 Regular / 阿里巴巴普惠体 Regular | 常规 |
| 悬疑/反差 | 得意黑 / 站酷酷黑 | 可倾斜、艺术化 |
| 品牌系列名 | 思源宋体 Medium | 优雅正式 |

## 6. A/B 测试建议

### 测试要素

| 测试维度 | 变量 A | 变量 B | 测试指标 |
|----------|--------|--------|----------|
| **标题风格** | 悬念钩子前置"离婚协议一签……" | 疑问句式"她真签了离婚协议？" | 点击率（CTR） |
| **人脸表情** | 微哭/悲伤表情 | 冷峻/无表情 | 点击率 + 完播率 |
| **色彩方案** | 暖调（甜宠粉） | 冷调（高级灰） | 点击率 |
| **标题位置** | 中下部（黄金分割） | 顶部（覆盖人脸以下） | 点击率 |
| **字体大小** | 大标题 96pt + 小副标题 | 中等标题 72pt + 详细信息 | 点击率 |
| **人物数量** | 单人封面 | 双人对话封面 | 点击率 + 转化率 |

### A/B 测试流程

1. **首轮测试**：每个标题变体制作 2-3 个封面版本
2. **投放量**：每个变体投放 ≥ 500 次曝光（小成本测试量）
3. **测试时长**：24 小时内出初步数据
4. **判定标准**：
   - 点击率差异 ≥ 10% → 选择胜出版本
   - 点击率差异 < 10% → 扩大样本量重测
5. **季度词库更新**：
   - 每月根据热点更新标题词库
   - 每季度淘汰 CTR 最低的 20% 标题句式
   - 统计各题材 TOP10 标题句式存档
