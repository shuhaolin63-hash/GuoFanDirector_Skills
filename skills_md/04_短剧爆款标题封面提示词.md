# 04 短剧爆款标题封面提示词

> **风格归零：郭帆视觉体系 x 短剧营销设计**
> 本提示词用于生成短剧的标题文案和封面视觉方案。
> 核心锚点：留白式悬念、信息克制、电影级海报质感、人设优先。

---

## 一、标题规则

### 1.1 核心原则：3秒钩子 + 留白悬念

郭帆式标题哲学：**"好标题不是告诉观众故事是什么，是让观众想知道故事是什么。"**

| 维度 | 规则 | 解释 |
|------|------|------|
| 信息量 | 给3分，留7分 | 标题只透露故事的冰山一角 |
| 情绪指向 | 不判断，只呈现 | 不要"虐恋"，不要"爆笑"——呈现一个状态，让观众自己感受 |
| 视角 | 从人物出发 | 标题是人物的一句话/一个动作/一个状态，不是创作者的概括 |
| 字数 | 单句≤15字 | 竖屏封面最多能舒适容纳15个字（含标点） |

### 1.2 标题类型

| 类型 | 公式 | 示例 | 说明 |
|------|------|------|------|
| **状态型** | [人物] + [一个正在进行的状态] | "他在等一个人" | 不说是谁，不等谁。给状态，不给答案 |
| **动作型** | [人物] + [一个中断的动作] | "她推开了那扇门" | 动作停在最关键的那一刻 |
| **物件型** | [一个物件] + [一个暗示] | "那封信没有署名" | 物件是叙事入口 |
| **反问型** | [一句没说完的反问] | "如果那天没有——" | 留白式断裂——让观众自己去接 |
| **时间型** | [一个时间点] + [一个事件] | "晚上十点零三分" | 精确的时间本身就制造悬念 |

### 1.3 标题字数规范

| 标题位置 | 最大字数 | 最佳字数 | 说明 |
|---------|---------|---------|------|
| 主标题 | ≤15字 | 8-12字 | 竖屏封面的视觉极限 |
| 副标题（可选） | ≤20字 | 10-15字 | 用于补充调性，字体缩小一号 |
| 封面文字 | ≤5字 | 2-4字 | 覆在封面图上，字体巨大，只出现一次 |

## 二、标题分类示例

### 甜宠题材

```
✅ 好的标题：
"他说好要回来的"
"她等的第三年"
"最后一封信"
"雨停之前"
"他没说出口的那句话"

❌ 坏的标题：
"腹黑总裁的独宠甜妻"（网文浮夸——信息全部丢出，没有悬念）
"霸道总裁爱上我之甜蜜暴击"（过长+评判性词汇）
"甜到齁！总裁大人轻点宠"（感叹号+情绪标签）
```

### 豪门题材

```
✅ 好的标题：
"她的嫁妆"
"协议的最后一条"
"那场婚礼没有宾客"
"他给的离婚协议"
"凌晨三点她签了字"

❌ 坏的标题：
"豪门虐恋：替身前妻带球跑"（网文套路标签）
"冷面总裁的契约娇妻"（浮夸+人物标签化）
"一夜惊喜：总裁的天价逃妻"（信息过载+低质网文感）
```

### 悬疑题材

```
✅ 好的标题：
"他看到了什么"
"监控录像第37秒"
"第三个失踪的人"
"她没有回家"
"那通电话是错的"

❌ 坏的标题：
"悬疑惊悚之死亡密码"（评判性词汇+太大了）
"烧脑神作！全程高能反转"（自夸——让观众自己发现）
"恐怖！深夜千万别点开"（命令式——低质封面套路）
```

### 都市情感题材

```
✅ 好的标题：
"合租室友"
"地铁末班车"
"他的婚礼她没去"
"欠了七年的晚安"
"外卖订单上的备注"

❌ 坏的标题：
"都市情缘：擦肩而过的爱"（空洞+评判）
"虐心催泪！看哭千万人"（情绪标签+自夸）
"职场精英的爱情抉择"（像论文标题+无悬念）
```

## 三、封面设计规则

### 3.1 人脸定位规则

与视频分镜相同，封面人脸定位遵循9:16竖屏视觉重心法则：

| 主体 | 位置规则 | 说明 |
|------|---------|------|
| 单人脸部 | 画面中央偏上1/3处 | 眼睛在水平中线偏上，头顶留白≤10% |
| 双人 | 错落排列，不并排 | 一个在中央偏上（主），一个在后或下方（辅） |
| 人物+环境 | 人物在下方1/3-1/2，环境在上方 | 不让人物淹没在背景中 |
| 手部/局部 | 居画面中央，留出想象空间 | 特写局部时确保构图完整 |

### 3.2 色彩方案与剧集色调绑定

封面色彩必须与短剧本身的色调模板一致。

| 题材 | 封面主色调 | 配色方案 | 文字颜色 |
|------|-----------|---------|---------|
| 甜宠 | 暖金/奶白/浅粉棕 | 奶油色系 + 一点暖光 | 深棕/黑（干净利落） |
| 豪门 | 冷白/青灰/暗金 | 冷色为主，局部金色点缀 | 白色（有重量感的粗体） |
| 悬疑 | 深蓝/紫/黑 | 暗色为主，一条冷色光 | 白色/冷银 |
| 都市 | 灰蓝/米白/暖黄 | 低饱和冷暖混搭 | 深灰/黑 |

### 3.3 封面文字规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 封面文字 ≤5字 | 图片上叠加的文字 | "第三年""那扇门""她" |
| 字体选择 | 无衬线粗体（黑体/思源黑体），不加描边 | 干净、硬实、不花哨 |
| 文字位置 | 画面下方1/3区域或画面中央留白处 | 不遮挡人脸 |
| 字号比例 | 主文字占画面宽度的60-70% | 大——但要留有呼吸空间 |
| 禁止 | 渐变/阴影/发光/艺术字/描边 | 纯白色或纯黑色文字 |

### 3.4 封面构图示例

**甜宠封面：**

```
┌─────────────────────────┐
│                         │
│    ▓▓▓▓▓▓▓▓▓▓▓▓         │  ← 暖金色天空/逆光
│    ▓▓▓▓▓▓▓▓▓▓▓▓         │
│      (人物侧脸剪影)      │  ← 人物在中央偏上，侧脸，逆光
│                         │
│                         │
│      ──────────          │
│     "第三年"             │  ← 文字在下方1/3，白色，粗体
│      ──────────          │
└─────────────────────────┘
```

**悬疑封面：**

```
┌─────────────────────────┐
│                         │
│    (一扇半开的门)        │  ← 视觉主体在画面中央
│      ┌────┐              │
│      │ ▓▓ │              │  ← 门缝中透出冷蓝光
│      └────┘              │
│                         │
│      ─────────           │
│     "监控"               │  ← 白色粗体，底部
│      ─────────           │
└─────────────────────────┘
```

## 五、负面禁止清单

```
❌ 网文式浮夸标题：赘婿/战神/大佬归来/隐世高手
❌ 感叹号标题：任何标题中出现"！"
❌ 自夸式标题："必看""爆款""热播""千万人都在看"
❌ 情绪标签词："虐心""甜炸""笑到哭""看哭了"
❌ 封面中的人脸被文字遮挡（鼻子/眼睛区域不可覆盖文字）
❌ 封面文字使用超过5个字
❌ 艺术字/描边字/渐变色文字/花体字
❌ 封面图有多个人物（超过2人——竖屏容纳不了）
❌ 封面使用白框/圆角/阴影/投影等装饰效果
❌ 封面logo/水印（保持画面干净）
❌ 封面色调和剧集色调不一致（甜宠封面做成冷色调）
❌ 空镜头封面——封面必须有人（或人的局部）或关键的叙事物件
```

## 六、标题+封面组合示例

### 示例1：甜宠剧

```
标题：他说好要回来的
封面文字：约定
封面图：女孩坐在窗边，逆光看向窗外，暖金色光线勾勒侧脸轮廓
色调：暖金柔光（cream + peach + warm gold）
文字位置：画面下方1/3，白色粗体
```

### 示例2：豪门剧

```
标题：协议的最后一条
封面文字：协议
封面图：一份文件特写，一只修长的手按在纸上，戒指反光
色调：冷白青橙（teal + cold white + amber accent）
文字位置：画面下方1/3偏左，白色加粗
```

### 示例3：悬疑剧

```
标题：监控录像第37秒
封面文字：37"
封面图：监控视角画面模糊，一个人影在走廊尽头
色调：蓝紫暗调（deep blue + near-black + cold cyan）
文字位置：画面下方1/3偏右，白色
```
