# README：郭帆导演风格短剧创作技能体系 — 调用说明

> **版本**：v1.0
> **体系定位**：基于郭帆导演视觉体系（短句硬实、冷克制+内热情感、细节写实主义、人设统一、留白式叙事）的短剧全流程创作工具集。

---

## 一、技能调用流程

### 步骤 1：定位题材 → 锁定色调模板

| 你选择的题材 | 对应的色调模板 | 对应的音频风格 |
|-------------|--------------|---------------|
| 甜宠/青春 | 暖金柔光（文件2-模板A） | 暖金配乐（文件3-4.2甜宠） |
| 豪门/商战 | 冷白青橙（文件2-模板B） | 冷工业配乐（文件3-4.2豪门） |
| 悬疑/惊悚 | 蓝紫冷霓虹（文件2-模板C） | 蓝紫压迫配乐（文件3-4.2悬疑） |
| 都市情感 | 混合——按情绪选择上述模板 | 按情绪混合 |

**操作**：翻开对应文件，将色调模板和音频风格复制到对应工具的提示词中。

---

### 步骤 2：创作剧本（使用文件1）

**输入工具**：GPT-4o / Claude / 豆包 / DeepSeek

**操作**：
1. 打开 `01_短剧剧本文本提示词（GPT_豆包_Claude）.md`
2. 将文件第1-6节（系统角色设定 + 六条原则 + 人物规则 + 台词规则 + 叙事节奏 + 负面清单）**整体复制**到AI的提示词框。
3. 在用户消息中给出：题材类型 + 核心人设（至少2个主角） + 第一集起始点（一句话）。
4. 得到初稿后，对照"负面禁止清单"（文件1-第六节）逐条检查。
5. 如需修改，直接要求AI按照原则重写。不要接受不符合"短句硬实"标准的输出。

**预期产出**：分集剧本（Markdown格式，按文件1-第七节的模板）。

---

### 步骤 3：生成视频分镜（使用文件2）

**输入工具**：Sora / 可灵 / 即梦 / Runway / Pika

**操作**：
1. 打开 `02_短剧分镜视频提示词（Sora_可灵_即梦_Runway）.md`
2. 从剧本中提取每一镜的关键信息。
3. 按文件2-第二节"四要素"为每个角色建立人物统一性描述。
4. 按文件2-第三节选择对应的色调模板，将模板后缀加入每段提示词。
5. 按文件2-第六节选择运镜方式，写入提示词。
6. 每段提示词前加上 `[CHARACTER]` 标签和四要素描述。
7. 生成后检查画面一致性，如人物漂移需重新抽卡或微调四要素描述。

**预期产出**：每镜一条视频生成提示词 + 生成的视频素材。

**效率建议**：批量生成同一角色的所有镜头提示词，一次性提交给AI视频工具，避免反复修改人物描述。

---

### 步骤 4：音频配音与音效设计（使用文件3）

**输入工具**：配音导演 / TTS合成工具 / 音效编辑软件

**操作**：
1. 打开 `03_短剧音频配音音效提示词.md`
2. 为每个角色建立"人物声音档案"（文件3-第五节模板）。
3. 按照剧本输出每集的"分场配音指导表"（文件3-第六节格式）。
4. 按题材选择配乐风格（文件3-4.2），配乐仅在关键情绪节点使用。
5. 每集至少安排一次30秒以上的"只有环境音+人声"段落。

**预期产出**：每集配音指导表 + 配音音频素材 + 音效素材 + 配乐方案。

---

### 步骤 4（并行）：标题封面设计（使用文件4）

**输入工具**：文案人员 / AI文案工具 / Midjourney / DALL-E

**操作**：
1. 打开 `04_短剧爆款标题封面提示词.md`
2. 按文件4-第一节规则创作3-5个候选标题。
3. 从中选出最佳标题，按文件4-第三节设计封面方案。
4. 确定封面文字的2-4字核心词。
5. 使用AI绘图工具（Midjourney等）生成封面图，提示词参考文件4-第四节。
6. 将封面文字叠加在图上——白色或黑色粗体，无衬线字体。

**预期产出**：1个主标题 + 1个封面图。

## 四、使用注意事项

### 4.1 风格一致性

- **整体性原则**：剧本风格决定视频风格，视频风格决定音频风格。如果剧本不符合郭帆风格（比如出现了长篇内心独白或甜宠金句），那么后续的视频和音频即使质量再高，整体也会失调。
- **色调绑定**：一旦在第1步选择了色调模板，文件2（视频）、文件3（音频配乐）、文件4（封面）都必须使用同一种色调。甜宠剧的视频不能突然出现冷色调镜头，悬疑剧的配乐不能突然变成暖色弦乐。

### 4.2 AI工具的限制与应对

- **人物形象漂移**：AI视频生成的最大问题。文件2的"四要素"是专门为此设计的解决方案。如果漂移仍发生，缩减单条提示词中的人物描述长度——元素越少，一致性越高。
- **AI配音的自然度**：目前TTS工具在"极简台词"（<=15字）场景下表现尚可，但在"留白式停顿"上几乎全部不合格。建议：关键情绪段落使用真人配音演员，非关键场景可使用TTS+后期EQ处理。

### 4.3 负面清单的使用

每个文件的末尾都有"负面禁止清单"。建议：
1. **创作前**通读一遍，建立禁忌意识。
2. **创作后**逐项检查输出，不通过的项目必须返工。
3. 负面清单不是建议，是"及格线"——有一项不合格就是不合格。

### 4.4 迭代建议

- 第一轮：聚焦文件1（剧本），打磨到负面清单全部通过后再进入下一步。
- 第二轮：使用文件2生成视频分镜，每集先出3-5镜测试，确认人物统一性和色调一致性后再批量生产。
- 第三轮：音频和封面同步进行。
- 每一轮完成后都必须回到文件1的"负面禁止清单"做整体检视。
