## 一、3秒强钩子规则

### 1.1 规则定义

短剧的核心流量逻辑：**前3秒决定留存率**。在用户划走的极限时间内，必须给出一个无法抗拒的理由，让其继续观看。

### 1.2 三种钩子类型

| 钩子类型 | 定义 | 例证 |
|---------|------|------|
| **视觉冲击钩子** | 以高反差、强色彩、极端画面瞬间捕获视线 | 车祸瞬间、扇耳光、跪地、摔东西、奢侈品特写 |
| **冲突信息钩子** | 以高信息差或强冲突迅速建立剧情张力 | "你被开除了"、"他不是你亲生父亲"、身份揭露 |
| **悬念钩子** | 以未完成的动作或未揭晓的信息制造必须看下去的理由 | "他到底是什么人？"、"三年前的那个夜晚到底发生了什么" |

### 1.3 3秒结构公式

```
第1秒 → 视觉冲击/悬念画面（让人停下来）
第2秒 → 冲突/危机/反转信息（让人想知道后续）
第3秒 → 钩子收束——必须看下去的理由（让人不划走）
```

**例证**：

> 第1秒：女人把离婚协议拍在桌上（视觉冲击）
> 第2秒：男人冷笑——"签字可以，但你肚子里那个，不是我的。"（冲突信息+反转）
> 第3秒：女人瞳孔收缩——她从未告诉任何人自己怀孕（悬念钩子）

### 1.4 郭帆开篇理念的对接

**郭帆原则**：危机前置，开篇即冲突。

**原话引用**（来源：混沌大学演讲）：

> "如果一段剧情删除后不影响人物成长和主题表达，则必须删除。"

**短剧对接**：短剧的开篇不需要"建立世界观"、"铺陈背景"、"逐格交代人物"。直接进入危机/冲突/悬念的高浓度信息带，用90秒讲完一场"微型电影"。

## 二、每15-20秒节奏波动规则

### 2.1 规则定义

短剧观众注意力的自然衰减周期约为15-20秒。因此，每15-20秒必须有一次节奏波动——反转、新信息、冲突升级、情绪转折——以重新捕获注意力。

### 2.2 标准节奏模板

| 时间节点 | 节奏要求 | 叙事功能 |
|---------|---------|---------|
| 0-3秒 | 强钩子 | 抓住注意力 |
| 3-15秒 | 建立情境，抛出第一个冲突 | 让观众理解"发生了什么" |
| 15-30秒 | 冲突升级/第一次反转 | 第一波节奏波动 |
| 30-45秒 | 信息揭示/认知颠覆 | 第二波节奏波动 |
| 45-60秒 | 高潮/结尾悬念 | 引导下一集 |

### 2.3 郭帆层层递进结构的对接

**郭帆原则**：

```
小冲突 → 升级危机 → 反转打破认知 → 结尾留白悬念
```

**短剧适配**：

| 郭帆层级 | 短剧节奏点 | 执行要求 |
|---------|-----------|---------|
| 小冲突 | 前3-15秒 | 一目了然的冲突，不需要背景交代 |
| 升级危机 | 15-30秒 | 冲突加剧，加入新变量 |
| 反转打破认知 | 30-45秒 | 观众预期被打翻 |
| 结尾留白悬念 | 45-60秒 | 制造"必须看下一集"的冲动 |

## 三、竖屏适配规则

### 3.1 信息纵向化原则

竖屏9:16画幅窄于横屏，所有构图必须围绕纵向展开。

| 横屏构图（16:9） | 竖屏构图（9:16） |
|----------------|----------------|
| 横向多人同框 | 纵向逐个切换特写 |
| 远景展现场景 | 中景+近景交替 |
| 全景建立空间 | 前景-中景-后景纵深替代 |
| 水平运镜平移 | 纵向推拉+上下跟随 |

### 3.2 前景/中景/后景纵深三层次

**核心原则**：用纵向层次替代横向展开。

| 层次 | 位置 | 内容 | 功能 |
|------|------|------|------|
| 前景 | 画面底部1/4 | 人物局部、道具、手部动作 | 制造临场感和代入感 |
| 中景 | 画面中部1/2 | 人物上半身或全身、主要对话 | 叙事主体，承载核心信息 |
| 后景 | 画面上部1/4 | 环境背景、光源 | 氛围营造、情绪定调 |

**例证**：一场办公室对峙戏
- 前景：桌上被拍响的解雇文件（冲击）
- 中景：女主管冷峻的面部表情（权力关系）
- 后景：落地窗外城市夜景（气氛烘托）

### 3.3 郭帆构图原则的竖屏迁移

**郭帆原则**（来源：美术指导郜昂）：

> "从概念设计到做成实体化的东西，有很多观众看不到的细节。我们在设计、制作时会尽量使这些细节合理一些。"

**竖屏迁移**：

- **减法构图**：画面中只保留推动叙事的核心元素，其余全部剔除
- **主体聚焦**：人脸固定画面上部1/3区域，核心肢体动作严格居中
- **细节特写**：道具/表情/动作的细节特写替代大全景，适配手机观看习惯
- **禁止无效信息**：禁止无效大全景、空景、浪费竖屏画幅

### 3.4 竖屏镜头景别比例铁律

| 景别 | 占比 | 使用场景 |
|------|------|---------|
| 近景 | 不低于60% | 短剧主力镜头，对话和情感表达 |
| 特写 | 20-30% | 情绪高潮、关键信息揭示、道具展示 |
| 中景 | 10-15% | 人物动作展示、关系建立 |
| 全景 | 不超过5% | 仅限建立场景关系的关键节点 |

## 四、连载留白规则

### 4.1 规则定义

短剧的连载属性要求：**每集结尾必须制造必须看下一集的冲动**。

### 4.2 五种留白类型

| 留白类型 | 定义 | 例证 |
|---------|------|------|
| **信息悬念** | 关键信息未完全揭示 | 一个秘密身份只揭露一半 |
| **决定悬念** | 角色面临十字路口，结果留至下集 | 主角拿起电话——打还是不打？ |
| **反转悬念** | 结尾处出现颠覆性反转 | 一直以为是好人的人，露出了真面目 |
| **危机悬念** | 角色陷入即将降临的危险 | 车冲向悬崖，画面戛然而止 |
| **时间悬念** | 以时间倒计时或预告压缩 | "三年前的那个晚上——"画面切黑 |

### 4.3 郭帆留白式叙事的对接

**原话引用**（来源：《流浪地球2》电影制作手记，剪辑叶潇畅）：

> "我对信息密度概念的理解，是画外未说的信息量。这种信息量可能像是绘画里的留白，是要通过自己的想象去补全的。"

**短剧对接**：

**关键信息不要一次说透，分多次揭示**。郭帆在《流浪地球2》中对图恒宇的叙事即是如此——他的决定在当下语境中是违法的、不可言说的，剪辑只保留了关键信息点，其余全部留给观众自行补全。

**在短剧中的应用**：
- 每集结尾将"最大悬念"留在画外
- 不给完整解答，给的是"大部分答案+一个新问题"
- 情绪高潮处适可而止，用画面替代文字解释

### 4.4 留白式结尾写作模板

```
[本集高潮场景] → [关键信息断裂] → [黑屏/画面定格] → [下集预告/文字提示]
```

**例证**：
> （他对面的人摘下了口罩）
> "三年前的那个人，就是我。"
> 她手中的杯子摔在地上，碎裂。
> 画面切黑——下集：他的真实身份，将揭开所有谜底。

## 五、短剧标题封面流量逻辑

### 5.1 标题公式

**核心原则**：标题必须在1秒内制造"信息差"——让观众觉得"我知道的比别人多想看点"。

| 标题类型 | 公式 | 例证 |
|---------|------|------|
| 身份反转型 | [他的真实身份竟是] + [身份名词] | "他捡了个垃圾，没想到是前首富的父亲" |
| 真相揭露型 | [你以为的] + [实际是反转] | "她以为是偶遇，其实是一场精心策划的局" |
| 极限情境型 | [极端条件] + [不可能的人物] | "结婚三年，老公没碰过我，直到他白月光回国" |
| 代价呈现型 | [一个选择] + [意外后果] | "她捐了一个肾，第二天发现配型成功的是仇人" |

### 5.2 封面设计原则

| 原则 | 内容 | 禁止事项 |
|------|------|---------|
| 人物聚焦 | 核心人物正面或半侧面，面部表情清晰可辨 | 多人杂乱的合影 |
| 情绪高度 | 情绪浓度最高的瞬间定格（愤怒、惊讶、哭泣） | 面无表情的证件照式封面 |
| 冲突暗示 | 画面暗示冲突关系（对峙、拥抱、拉扯） | 无关系的风景或空镜 |
| 信息密度 | 封面本身就在"讲故事" | 纯文字+纯色背景 |

### 5.3 郭帆"观众看得懂"原则在封面的应用

**原话引用**（来源：《流浪地球2》电影制作手记）：

> "我们的唯一标准就是观众看得懂，这个最重要。"

**短剧标题/封面对接**：
- 标题必须在1秒内被理解——不是看完标题，是扫一眼就知道讲什么
- 封面人物表情必须"一眼可读"——愤怒、委屈、惊讶、不屑
- 禁止文艺晦涩的标题——"她"、"往事"、"那一年"——这类标题在短剧赛道是致命的
- 信息优先：标题用最直白的语言说出最核心的冲突或反转

## 六、与郭帆叙事节奏体系的结合点

### 6.1 综合节奏对照表

| 时间 | 短剧流量规则 | 郭帆叙事体系 | 执行要点 |
|------|------------|------------|---------|
| 0-3秒 | 强钩子 | 危机前置、开篇即冲突 | 用最少的铺垫，直接进入冲突 |
| 3-15秒 | 建立情境 | 人物驱动，信息密度优先 | 只保留推动剧情的信息点 |
| 15-30秒 | 第一次节奏波动 | 冲突升级 | 加入新变量，改变力量对比 |
| 30-45秒 | 信息揭示/认知颠覆 | 反转打破认知 | 让观众意识到"之前的理解是错的" |
| 45-60秒 | 结尾悬念 | 留白式叙事 | 关键信息断裂，制造追剧冲动 |

### 6.2 郭帆节奏四段式 vs 短剧流量节奏

| 郭帆结构 | 对应短剧节奏 | 核心技巧 |
|---------|------------|---------|
| 小冲突 | 3秒钩子+情境建立 | 不需要背景交代，直接进入冲突 |
| 升级危机 | 第一波反转（15-30秒） | 加入新变量，改变力量对比 |
| 反转打破认知 | 第二波反转（30-45秒） | 观众预期被打翻 |
| 结尾留白悬念 | 连载钩子（45-60秒） | 关键信息断裂，制造"必须看下一集"的冲动 |

### 6.3 郭帆"删繁就简"理念在短剧的应用

**原话引用**（来源：《流浪地球2》电影制作手记，剪辑指导张嘉辉）：

> "怎么掌握这个分寸，对于我来说很简单：看完以后看得懂吗？如果看得懂，没问题；如果看不懂，那就有问题了。"

**短剧硬性规则**：

**每集信息筛选题**——每个场景、每句台词、每个角色，回答以下问题：
1. 它推动剧情了吗？
2. 它改变人物状态了吗？
3. 它制造/升级冲突了吗？
4. 它揭示人物性格了吗？
5. 它埋下后续伏笔了吗？

**如果以上全部否，删除。没有例外。**

### 6.4 郭帆"不为题材所迷只向人物用力"的短剧意义

短剧最致命的陷阱：为了"流量"牺牲人物逻辑——角色做出不符合性格的行为，只为了制造爽点或打脸。

**郭帆提醒**：真正让观众追下去的永远是人物命运，不是爽点套路。人物驱动故事，而非概念驱动。短剧的"爽"必须建立在"这个人这样做合理"的基础上。
