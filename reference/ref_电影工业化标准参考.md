## 一、工业化的定义与核心逻辑

### 1.1 郭帆的定义

> **"工业化的核心就是标准化，然后才可以量化、细分，可被分配，提高效率。这是工业化的基础逻辑。"**
> ——来源：《中国经济周刊》专访，2019年2月

这五个步骤构成了工业化闭环：

```
标准化 → 量化 → 细分 → 可分配 → 提高效率
```

### 1.2 工业化的本质：基础设施+工具包

**原话引用**：

> "工业化其实是一个基础设施建设过程，它是一个工具包，只有把所有的方法论都总结好了，打包好了，它才能够高效。"
> ——来源：《流浪地球2》电影制作手记

> "工业化是什么呢？它特别像画家画画的笔和纸，它是一个工具。但我们没有那么趁手的工具，所以我们没有办法像好莱坞的创作者一样，他们脑海中想到了什么，就可以变成镜头。"
> ——来源：混沌大学演讲，2019年

### 1.3 当前中国电影工业化的短板

**郭帆的评价**（来源：《流浪地球2》电影制作手记）：

> "在整个剧组里，目前工业化水平最高的是摄影部门，主要是因为他们的设备是跟国际同步的，设备使用的流程和规范是最标准的。第二梯队是视效，视效流程的工业化程度也很高，也是因为技术是这个行业的核心。其他没有技术加持、主要依赖沟通的部门工业化程度普遍有待加强，比如导演部门和制片部门。"

**最大困难**不是技术，而是观念：

> "有很多人不愿意使用更新的软件和工具，更愿意用纸跟笔。而且大部分人的观念是觉得这件事情并没有什么。但是用纸跟笔记录，在一时可能是快的，但积累出100天或200天的工作之后，你会发现所有的信息都很容易丢失，并且难以查询。"
> ——来源：《中国经济周刊》专访，2019年

## 二、从流浪地球1到2的工业化进化

### 2.1 道具制作的飞跃

| 项目 | 《流浪地球》（2019） | 《流浪地球2》（2023） |
|------|-------------------|-------------------|
| 宇航服 | 少量制作，手工为主 | 一千一百多个零件，批量化生产 |
| 外骨骼 | 少量手工制作 | 维塔工作室开发5套主演版，中方复制改装数十套轻版 |
| 道具数量 | 相对简单 | 5310张概念设计，9.5万件道具 |
| 特效镜头 | 约2000个 | 3300个（一说三千多个） |
| 参与视效公司 | 少量 | 六十几家 |
| 生产工具 | 传统手工 | 3D打印+数字车床+激光雕刻 |

**郭帆的原话**：

> "原来好莱坞是按产品生产逻辑去做道具，所以在最初的时候，建立所有模型都是有内构的。你会发现当你借他们的力返回来做这些事情时候，瞬间就超过了好莱坞的那些人，就是因为我们生产能力太强了，是因为国家工业体系的建立，让我们这件事情完整。"
> ——来源：人民日报专访，2023年

### 2.2 生产线的建立

> "我们在现场拉出一条生产线，将3D打印技术和数控车床应用到电影生产中。"
> ——来源：《流浪地球2》圆桌对谈，2023年2月

> "技术的应用融合和生产工具的提升，让电影拍摄的效率大大增加。"

### 2.3 从做不出宇航服到批量生产15套

**第一部**：物理特效公司经验不足，"报价就很低，因为没经验，但真正去实操时，就发现完成不了。"制片人龚格尔甚至"把车也卖了"来填补道具制作缺口。（来源：混沌大学演讲）

**第二部**：
- 宇航服1100多个零件，软质板500多个，硬质零件500多个
- 外部硬质部分属于高阶曲面，和汽车一样
- MDI工作室攻关整条工作流程
- 同时批量生产15套不同规格的宇航服

### 2.4 视效流程的工业化进化

**第一部**：线性工作流，一个人负责某一个模块。

**第二部**：
- 从线性工作流到全员"全栈艺术家"
- 使用Blender实现快速渲染和迭代
- 规避按流程分板块的责任问题
- 采用USD（Universal Scene Description）作为数据流开发新Pipeline
- 整个Pipeline框架重构
- 创造性地加入动态版本管理

### 2.5 后期制作数据化管理升级

| 维度 | 第一部 | 第二部 |
|------|-------|-------|
| 沟通方式 | 传统沟通 | Shotgun项目管理+Rocket Chat聊天系统+RV集成讨论 |
| 文件管理 | 手动传输 | 阿里云OSS自动化资产分享和文件回传 |
| 外包管理 | 传统对接 | Vue+Electron开发外包客户端，Excel模板批量提交 |
| 数据格式 | 无统一标准 | USD格式统一数据流 |

## 三、预拍摄流程

### 3.1 完整流程链条

```
概念设计 → 静态分镜 → PreViz动态故事板 → 虚拟拍摄 → 实拍
```

郭帆强调这一流程的核心价值：**把创新部分前置到筹备阶段**。

**原话引用**（来源：《中国经济周刊》专访）：

> "（拍第一部时）记得我第一天坐在监视器前是坐不住的……我发现，其实做导演不是一个简单的、只是创作的事儿，他是一个特别综合性的工作，包括人际关系的处理、管理以及拍摄现场的控制。因此你会发现，创作在现场其实并不是最关键的部分。"

> "（在第二部时）把更多的创新部分前置到了筹备阶段。在实际的拍摄阶段不参与更多的创意，而是把精力留在处理那些具体问题上。"

### 3.2 PreViz动态故事板

**来源**：《流浪地球2》电影制作手记

- 预见科幻：PreViz动态故事板
- 第一部片尾动画分镜头画稿展示了从分镜到成片的完整流程
- PreViz帮助导演和团队理解镜头运动和节奏

### 3.3 虚拟预拍摄（全片预演）

**原话引用**（来源：人民日报专访，2023年）：

> "我们用了大概两个多月的时间，在一个虚拟的空间里去拍摄整部戏。演员全部穿着动作捕捉衣，然后他们从头演一遍，所有调度也都彩排一遍。在画面中可以看到这个景是大是小，从这可以调整，全组的所有人都非常清楚我们今天要拍什么，因为他看到了。"

**意义**：《流浪地球2》引入了"全程预拍摄"概念。创作团队通过分镜绘制、动态预演、虚拟拍摄三种方式，在正式开拍前完成部分影片的模拟拍摄和剪辑制作。优化实拍方案，大幅节约筹备、调度和拍摄的时间。

### 3.4 分镜设计原则

**来源**：《流浪地球》电影制作手记，概念设计/故事版指导张勃

| 原则 | 内容 |
|------|------|
| 叙事清晰 | 分镜必须清晰传达叙事意图 |
| 节奏控制 | 通过分镜控制影片节奏 |
| 视觉冲击 | 关键场景的分镜要具有视觉冲击力 |
| 技术可行 | 分镜设计要考虑实际拍摄的可行性 |

**第一部数据**：团队做了8000多格分镜，"里边还会有配音、配乐"。（来源：混沌大学演讲）

## 四、道具生产标准

### 4.1 全流程标准：从螺丝孔径到分模线

**原话引用**（来源：《流浪地球2》电影制作手记，美术指导郜昂）：

> "从概念设计到做成实体化的东西，有很多观众看不到的细节。我们在设计、制作时会尽量使这些细节合理一些。比如螺丝孔径的大小，包括应用什么螺丝，什么样的分模线会出现在某一个单体上。看到一个录音笔，录音笔的出音孔或收音的孔位大概采取怎样的阵列点打孔方式；正面和后壳之间的分缝，以及分缝之间的螺丝孔。还有一些结构性的东西。"

### 4.2 道具工业化三原则

| 原则 | 内容 | 例证 |
|------|------|------|
| 细节合理性 | 每个细节都有物理和功能依据 | 螺丝孔径大小、分模线位置、收音孔阵列方式 |
| 可批量化生产 | 模块化设计、拆解后批量生产 | 宇航服1100多个零件全部模块化 |
| 真实感至上 | 使用痕迹、磨损、灰尘等真实细节 | 每一件道具都有使用感，不是"崭新出厂" |

### 4.3 关键道具数据

| 道具 | 关键数据 | 制作团队 |
|------|---------|---------|
| 宇航服 | 1100+零件，软质板500+，硬质零件500+ | MDI工作室 |
| 外骨骼 | 80公斤，穿戴需多人配合约1小时 | 维塔工作室+中方团队 |
| 运载车 | 剧本中75%情节围绕其发生 | 中央美院交通工具设计系+希娜魔夫 |
| 气囊球 | 3米直径亚克力硬球 | 自制 |
| 数字生命备份卡 | 前端思维芯片+后端记忆芯片 | 中科院数字生命研究所（设定） |
| 爆点火核心 | 只组装面向镜头的一半，重心偏移靠人配平吊装 | 制片主任李宝泉"上人！" |

### 4.4 工业级制造手段

- 3D打印：道具原型快速成型
- 数字车床：精密零件加工
- 激光雕刻：细节纹理制作
- 高阶曲面建模：宇航服外部零件（与汽车同级标准）

### 5.1 项目管理工具链

| 工具 | 用途 | 说明 |
|------|------|------|
| Shotgun | 项目管理和任务跟踪 | 任务列表显示关键信息，便于外包公司查看 |
| Rocket Chat | 即时通讯 | 各部门沟通协作 |

| 阿里云OSS | 文件存储与分发 | 资产分享和文件回传自动化 |

### 5.2 外包管理流程

- **客户端开发**：使用Vue+Electron开发外包专用客户端
- **批量提交**：允许外包使用Excel模板大批量提交小样
- **自动化分发**：利用阿里云OSS接口实现资产自动分发

### 5.3 视效审阅标准

每个镜头从开始到最终输出，平均经历：
- 3至5次沟通
- 4至6次细化
- 5至8次反馈
- 10次以上审阅

### 5.4 剧本数据化

**原话引用**（来源：《中国经济周刊》专访）：

> "如果没有这种标准化的剧本，我们就很难做到将它数据化，并且把它变成数据库，再和后续的各个部门间的工具去连接。所以就很难做到更加细致的分工和管理。"

## 七、对短剧生产流程的工业化启示

### 7.1 标准化 → 量化 → 细分 → 可分配 → 提高效率

**短剧应用路径**：

| 工业化步骤 | 短剧对应措施 |
|-----------|------------|
| 标准化 | 统一剧本格式、分镜模板、后期质检清单 |
| 量化 | 每集时长（60-180秒）、每集场景数（3-5个）、每集反转数（1-2个） |
| 细分 | 剧本组→拍摄组→后期组→质检组，每环节专人负责 |
| 可分配 | 并行生产多集，各集之间独立流程 |
| 提高效率 | 模板复用+经验积累+工具自动化 |

### 7.2 预拍摄流程的短剧适配

| 郭帆预拍流程 | 短剧简化版 |
|------------|-----------|
| 概念设计 | 关键场景设计图/参考图 |
| 静态分镜 | 每集5-8张关键帧分镜 |
| PreViz动态故事板 | 核心动作场景简易动画预览 |
| 虚拟拍摄 | 关键特效场景预演 |
| 实拍 | 标准化拍摄 |

### 7.3 道具/场景标准化的短剧应用

- **场景复用**：建立场景模板库（办公室、豪宅、学校、病房等），同一场景不同剧集可微调后复用
- **道具标准化**：建立常用道具清单和获取渠道，减少每次重新设计的成本
- **服装模块化**：人物造型绑定公式，同一角色全程锁定，减少换装成本

### 7.4 数据化管理的短剧版本

| 工具 | 短剧替代方案 | 用途 |
|------|------------|------|
| Shotgun | 飞书/钉钉项目管理 | 任务分配和进度跟踪 |
| Rocket Chat | 微信群/飞书群 | 即时沟通 |
| 云存储 | 阿里云OSS/腾讯云COS | 素材和成片管理 |
| 版本管理 | 文件名规范化+v1/v2/v3标记 | 避免素材混乱 |

### 7.5 工业化核心启示清单

1. **剧本标准化是工业化的第一块砖**：统一格式→数据化→数据库→部门间工具连接
2. **工具包思维**：每次创作后复盘，补充模板/清单/流程文档，让下一次更高效
3. **流程决定质量下限**：标准流程确保"批量生产有平均水准"，而非每次从零开始
4. **数据化管理降低沟通成本**：信息可追溯、可查询、可分配，减少口头传递的遗漏
5. **预拍摄前置问题**：在实拍前发现并解决问题，节省实际拍摄的时间和资源
